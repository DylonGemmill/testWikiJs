Lo sviluppo di estensioni Joomla si fa tendenzialmente con Eclipse e Phing. Eclipse è un IDE, mentre Phing è un installer.

Installare eclipse, quindi andare in Help > Install new software. Aggiungere come url 

http://download.eclipse.org/releases/helios/

e premere il pulsante di aggiunta. Appena popolata la parte inferiore con l'elenco dei pacchetti disponibili espandere "Programming languages" e selezionare PHP. Proseguire la fase guidata ed al termine partirà in background l'installazione. Completato il download una finestra di dialogo proporrà il riavviodi Eclipse.

 

apt-get install php-pear
pear channel-discover pear.phing.info
pear install phing/phing
