Collegarsi al pannello amministrativo del sito web ed andare in Componenti > Akeeba Backup. Premere il pulsante 

Se possibile immettere una descrizione (ad esempio: Prima di aggiunta nuovi prodotti, o Prima di aggiornamento).

Premere "Backup Now" ed attendere la conclusione del salvataggio, senza chiudere il browser o visitare altre pagine nella stessa finestra.

 

Completata l'operazione collegarsi via FTP e scaricare il file generato in una posizione per la conservazione.

 

In genere si consiglia di effettuare il backup ogni 7/15gg, e di conservare gli ultimi quattro file.
