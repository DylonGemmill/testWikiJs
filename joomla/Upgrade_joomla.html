Metodo n°2: Aggiornamento SemiAutomatico  se si è in dubbio sul percorso della cartella andare dal pannello amministratore:

sito->configurazione globale->sistema Percorso Log * modificandolo opportunamente con la cartella tmp

 

Risoluzione di eventuali problemi:

 

Upgrade Joomla alla versione 1.7.3

Dettagli

 

Cercando di aggiornare Joomla dalla versione 1.7.0 e 1.7.1 alla 1.7.3 vi sarà capitato di incappare nell'errore -1 copia fallita.

 

questa è la mia soluzione:

 

  1. modificare il file /libraries/joomla/filesystem/folder.php (fate prima una copia di sicurezza del file).
    andate alla riga 137 e e modificate la riga 
    return JError::raiseError(-1, JText::_('JLIB_FILESYSTEM_ERROR_COPY_FAILED'));
    con
    return JError::raiseError(-1, $dfid);
  2. Abilitate il debug dal backend di Joomla.
  3. controllate che le cartelle tmp e log abbiano i permessi di scrittura
  4. procedete all'upgrade come indicato su joomla.it
  5. se non andrà a buon fine, ora vi dirà qual'è il file che non riesce a sovrascrivere, modificate i permessi e riprovate finchè non vi darà più l'errore.
  6. se tutto è okay allora utilizzando la copia di backup di folder.php rimettetelo in libraries/joomla/filesystem
