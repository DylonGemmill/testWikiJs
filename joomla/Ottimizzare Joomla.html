• disabilitare i componenti, moduli e plugin non utilizzati
  • abilitare il caching (Sito > Configurazione > Sistema)
  • mantenere il database ottimizzato, via sql o tramite estensioni apposite (come OptimizeTables)
  • abilitare la compressione gzip delle pagine (Sito > Configurazione > Server)
  • abilitare il plugin System - Cache in Estensioni > Gestione plugin
  • se si usa un componente per SEF, abilitare la sua cache interna (ad es. in sh404sef, Configurazione > Avanzato)
  • se non altrove previsto, abilitare il caching via .htaccess, come descritto in questo articolo
  • installare il plugin CssJsCompress (verificare come funziona con VirtueMart, non l'ho testata bene!)
  • aggiungere la riga FileETag None nel file .htaccess
  • aggiungere gli header expire in .htaccess (vedi qui)
