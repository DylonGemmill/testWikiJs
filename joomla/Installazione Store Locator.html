Store Locator è una estensione per Joomla che consente di creare una lista di negozi, e visualizzare una mappa con ricerca sul frontend. Per ogni rivenditore è possibile specificare Nome, Indirizzo, sito web, email, twitter, facebook, oltre ad altri cinque campi personalizzabili.

L'installazione si esegue in pochi passaggi:

  1. Scaricare l'ultima versione del plugin dal sito internet (richiede sottoscrizione!). In allegato a questo documento l'ultima versione disponibile ad oggi
  2. Installare il componente da Estensioni > Installa / disinstalla
  3. Andare in Componenti > Store locator
  4. Premere il pulsante Preferenze
  5. Inserire l'API Key di Google Maps per il sito (se non si ha ancora crearla qui)
  6. Modificare le unità in km (units)
  7. Modificare lo stato base in Italia (base country)
  8. Rimuovere il link a Sysgenmedia (Show Sysgen Media Link)
  9. Se installato su un sistema dove è presente sh404sef, aggiungere lo script com_storelocator.php di regole per la creazione degli url SEF (in allegato) alla directory administrator/components/com_sh404sef/plugins/system
