Per rendere un foglio di stile importato ed editabile da jvframework occorre metterlo nella directory

templates/jv-framework/themes/MIOTEMA/css

e poi aggiungerlo all'elenco dei css da importare. Tale elenco si trova nel file

components/com_jvframework/basethemes/layouts/head.xml

Per aggiungerlo, modificare il tag relativo inserendo il nome del file (senza l'estensione css) separandolo con una virgola da quelli già presenti, esempio:

--- head.xml    2011-10-10 16:02:39.600743817 +0200
+++ head.xml.orig       2011-10-10 16:14:09.539514032 +0200
@@ -40,7 +40,7 @@
         
         <!-- Add style to head-->
         <stylefile  names='typo' path='base_theme' />   
-        <stylefile  names='layout,css3,k2' media='' path='' />   
+        <stylefile  names='layout,css3' media='' path='' />   
         
         <!-- Stylesheet from framework addon -->
         <addonstyle/>

per aggiungere il file templates/jv-framework/themes/MIOTEMA/css/k2.css
