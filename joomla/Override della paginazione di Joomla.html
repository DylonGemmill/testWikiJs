La paginazione standard di Joomla è gestita dal file libraries/joomla/html/pagination.php

 

In questo file è presente una classe che si occupa di gestire la paginazione. Il rendering della stessa è gestito da alcuni metodi, e non da file template come per le altre parti di Joomla. Tuttavia è possibile effettuare l'override dei soli metodi che ci interessano per avere una visualizzazione della paginazione ad hoc.

 

Creiamo quindi il file html/pagination.php all'interno del nostro template. Questo file non dovrà contenere nessuna classe, ma solo i metodi della paginazione su cui si vuole effettuare l'override (con una piccola accortezza sul nome della funzione). Se per esempio vogliamo effettuare l'override del metodo _list_render, nel nostro file dichiareremo la funzione pagination_list_render. In pratica va dichiarata la funzione con nome composto da pagination + nome del metodo di cui fare l'override.

 

Di seguito un esempio di file pagination.php

<?php defined('_JEXEC') or die;

function pagination_list_render($list)
{
  // Reverse output rendering for right-to-left display.
  $html = '<ul>';
  foreach ($list['pages'] as $page)
  {
    $html .= '<li><p>' . $page['data'] . '</p></li>';
  }
  $html .= '</ul>';

  return $html;
}

 

La chiamata nel template per visualizzare tale lista è

<?php echo $this->vmPagination->getPagesLinks (); ?>
