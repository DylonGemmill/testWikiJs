La stampante è supportata da HPLIP, ma la versione presente in lucid non è abbastanza recente, occorre usare quella dal sito hpopensource.com.

Installare le dipendenze dello script di installazione:

apt-get -y install python-gobject

poi il sw di HP chiederà altre dipendenze.

Verrà salvato un file .run, da lanciare con sh, ad esempio:

sh hplip-3.12.11.run

Se ci si trova su un desktop fare pure l'installazione automatica, altrimenti su un server scegliere quella manuale, per poter selezionare i componenti (sampa, scanner, fax).

Il programma si occuperà di scaricare le dipendenze necessarie e compilare i software di HPLip.

 

Se la stampante è via rete selezionare NO quando viene chiesto se installare la stampante, quindi lanciare invece il comando:

hp-setup -i IP_STAMPANTE
