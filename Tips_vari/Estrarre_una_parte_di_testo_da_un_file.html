Con awk è possibile estrarre una parte di un file di testo. Ad esempio da un dump MySQL è possibile estrarre la creazione e le insert di una tabella, o di un database, cosa che con dump grossi e vi non è fattibile.

Ad esempio per estrapolare la creazione di una tabella e le relative insert andiamo a prendere tutto il testo racchiuso tra le due righe Table structure for table `...`.

 

Il comando è il seguente:

awk '/Table structure for table `table1`/,/Table structure for table `table2`/{print}' fileinput > fileoutput
