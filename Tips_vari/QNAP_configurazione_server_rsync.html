Andare in Impostazioni > Server di backup, quindi nella prima scheda abilitare la voce Abilita backup da un server remoto al server locale, ed anche la successiva Consenti al server remoto di eseguire il backup dei dati sul nas. Nei campi di input inserire un username ed una password, da usare poi. Quando fatto premere Applica.

Il server è già configurato, ed avrà accesso alle stesse condivisioni create per gli altri servizi.

Sul server da cui si desidera fare il backup creare un file dove memorizzare la password inserita sopra (ad esempio /etc/rsync_password) e dare i permessi 600.

Per fare un backup da remoto verso la share backup usare il comando

rsync --password-file=/etc/rsync_password /dati utente_rsync@IP::backup/dati/

.
