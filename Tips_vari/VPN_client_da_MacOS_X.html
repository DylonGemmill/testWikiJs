Per MacOS X esiste un client grafico molto bello che si chiama TunnelBlick.

Scaricarlo dal sito ed installarlo.

La configurazione può essere fatta a partire da un file .ovpn (file di OpenVPN di Windows) oppure manualmente.

Nel caso di importazione, prima di effettuarla rimuovere il percorso del file davanti al certificato (deve esserci solo il nome file), e rimuovere tutto il nome file sull'opzione auth-client-config. Quindi importare il file insieme al .pem nella stessa posizione, che dovrebbe essere aperta dal wizard di configurazione.

 

Fatte queste operazioni la VPN è facilmente avviabile dopo aver lanciato TunnelBlick.
