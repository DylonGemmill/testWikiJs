Con Firefox 3.6 il javaplugin da linkare nella cartella dei plugins di firefox non è più libjavaplugin_oji.so ma libnpjp2.so

Nessun pacchetto di Ubuntu sembra contenere quel file, occorre quindi scaricare java dal sito di Oracle: http://java.com/it/download/

Scompattare il file in una directory a piacere (per esempio /opt) e linkare il file:

 

# mv jre-6u23-linux-i586.bin /opt
# sh jre-6u23-linux-i586.bin
[ ... accettare certificato, estrazione, ecc ... ]
# ln -s /opt/jre1.6.0_23/lib/i386/libnpjp2.so /usr/lib/mozilla/plugins/libnpjp2.so

 

Nell'esempio sopra il plugin viene linkato in /usr/lib/mozilla/plugins (tutti gli utenti del pc potranno usufruire del plugin), per fare ciò occorrono i permessi di root; in alternativa è possibile eseguire l'installazione solo per l'utente corrente scompattando il bin di java nella propria home e linkando il plugin in ~/.mozilla/plugins/

ATTENZIONE:
- i nomi dei file si riferiscono alla versione corrente di Java, nel vostro caso potrebbero essere diversi.
- con Firefox 3.6 occorre avere la versione di Java 6 update 10 o superiore. Non fatevi ingannare dal nome della directory di destinazione (jre1.6.0_23) quello che fa fede è il nome del bin (jre-6u23-linux-i586.bin).
