Un tool molto comodo per visualizzare le tabelle di un database access è mdbtools:
http://mdbtools.sourceforge.net/

sudo apt-get update
sudo apt-get install mdbtools

Permette di avere l'elenco delle tabelle salvate nel database (mdb-tables) e anche esportare i dati di una determinata tabella (mdb-exports, attenzione a rendirezzare l'output! )

https://linux.die.net/man/1/mdb-export
