Installare le dipendeze

aptitude install cmake libsqlite3-dev build-essential libsmbclient-dev liblog4c-dev libssh-2-dev

Scaricare e compilare iniparser

cd /usr/src
wget http://ndevilla.free.fr/iniparser/iniparser3.0b.tar.gz
tar xzf iniparser*
cd iniparser*
make
cp libiniparser.* /usr/local/lib/
ldconfig

Scaricare e scompattare http://www.csync.org/ in /usr/src

cd /usr/src
mkdir csync-build
cd csync-build
ccmake ../csync-0.44.0

Impostare il parametro INIPARSER_INCLUDE_DIR a /usr/src/iniparser3.0b/src. 

Premere c per configurare e g per generare il makefile.
Tornati al prompt lanciare

make 
make install 
ldconfig
