Faketime è una libreria che permette di modificare il tempo restituito dal sistema quando richiesto dai vari programmi, senza effettivamente modificare l'ora di sistema.

E' disponibile qui: https://github.com/wolfcw/libfaketime/archive/master.zip, scaricare come zip è caricare sul sistema desiderato

Installazione:

unzip libfaketime-master.zip
cd libfaketime-master
make
sudo make install
export LD_PRELOAD=/usr/local/lib/faketime/libfaketime.so.1

Modificare il tempo restituito quando si lanciano programmi dalla cli, ci sono due opzioni:

  1.  Creare un file .faketimerc nella home e inserire l'ora com '@2012-12-21 12:12:12'
  2. Creare una variabile d'ambiente export FAKETIME= '@2012-12-21 12:12:12'

Modificare il tempo restituito in apache2, quindi per i siti web che girano sotto apache2:

Aggiungere in fondo al file /etc/apache2/envvars

export FAKETIME="2008-01-01 00:00:00"
export LD_PRELOAD='/usr/local/lib/faketime/libfaketime.so.1'

Riavviare apache2:

sudo /etc/init.d/apache2 restart
