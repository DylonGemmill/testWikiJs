A tale scopo ci viene incontro un'estensione che fa da "wrapper" per TCPDF:

http://www.yiiframework.com/extension/tcpdf/

Scaricare l'estensione e posizionarla nella directory delle estensioni di Yii (protected/extensions). L'estensione contiene solo il file che fa da wrapper, TCPDF va scaricato a parte. La cartella creata dell'estensione sarà quindi protected/extensions/tcpdf.

Scaricare TCPDF dal sito ufficiale (http://www.tcpdf.org/) ed estrarlo in una directory a piacere. Prelevare quindi solo i file necessari (in modo da non appensantire l'applicativo). Tali file comprendono i file nella directory principale, e le cartelle config, fonts e images (tralasciando quindi documentazione, cache ed esempi) e vanno messi nella directory protected/extensions/tcpdf/tcpdf.

L'autore consiglia di aggiornare spesso TCPDF, dato che e' in continuo sviluppo.

A questo punto è possibile creare un pdf in qualsiasi punto della nostra applicazione. Una volta istanziato l'oggetto pdf, le funzioni a nostra disposizione sono le stesse di TCPDF; ecco un esempio:

<?php
$pdf = Yii::createComponent('application.extensions.tcpdf.ETcPdf', 'P', 'cm', 'A4', true, 'UTF-8');
$pdf->SetCreator(PDF_CREATOR);
$pdf->SetAuthor("Nicola Asuni");
$pdf->SetTitle("TCPDF Example 002");
$pdf->SetSubject("TCPDF Tutorial");
$pdf->SetKeywords("TCPDF, PDF, example, test, guide");
$pdf->setPrintHeader(false);
$pdf->setPrintFooter(false);
$pdf->AliasNbPages();
$pdf->AddPage();
$pdf->SetFont("times", "BI", 20);
$pdf->Cell(0,10,"Example 002",1,1,'C');
$pdf->Output("example_002.pdf", "I");
