Modificare il default controller di un modulo

Nella classe del nostro modulo, aggiungere l'attributo pubblico defaultController e settarlo col nome del controller desiderato. Esempio:

//inside protected/modules/admin/AdminModule.php
class AdminModule extends CWebModule
{
  //goes to TaskController instead of DefaultController
  public $defaultController = 'Task';

Aggiungere utilizzare modelli in moduli differenti

Yii, di default, non permette di utilizzare il modello di un modulo all'interno di un altro modulo. Per poterlo fare, occorre importare il modello nel secondo modulo. Per fare ciò, editare la funzione init del nuovo modello (file protected/modules/mymodule/MymoduleModule.php) e aggiungere il modello all'array relativo. Esempio, se volessi aggiungere tutti i modelli del modulo mymodule2, ecco come dovrà essere l'inizio del file MymoduleModule.php:

<?php

class MymoduleModule extends CWebModule
{
    public function init()
    {  
        // this method is called when the module is being created
        // you may place code here to customize the module or the application

        // import the module-level models and components
        $this->setImport(array(
            'mymodule.models.*',
            'mymodule.components.*',
            'mymodule2.models.*',
        ));
    }
