PHPMyFAQ possiede già un plugin per l'autenticazione HTTP, che però funziona solo con il classico metodo username/password.

A partire da quel plugin ne è stato scritto un'altro che accetta anche l'assenza della variabile contenente la password.

Il file AuthShib.php in allegato a questo documento va copiato nella cartella /path-pmf/inc/PMF_Auth/.

Per abilitare il metodo di autenticazione però, bisogna modificare manualmente il file index.php in quanto non esiste ancora un modo per attivare le diverse autenticazioni in maniera meno invasiva. In allegato il file patch da applicare all'originale, dopo averne fatto un backup

 

cd /path/di/pmf/
cp index.php index.php.ORIG
patch -i /dir/del/patch/file
