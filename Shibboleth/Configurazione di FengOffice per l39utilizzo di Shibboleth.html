FengOffice non supporta l'autenticazione HTTP ne esistono plugin che ne aggiungono le funzionalità. Per questo è stato necessario modificare il file di controllo degli accessi, aggiungendo il controllo sulla variabile $_SERVER['REMOTE_USER']

Per applicare le modifiche si può utilizzare il file patch presente in allegato a questo documento.

Spostarsi nella cartella in cui è situato il file in questione e fare un backup del file originale

cd /path/di/feng/application/controllers/
cp AccessController.class.php AccessController.class.php.ORIG

 A questo punto applicare la patch al file

patch -i /dir/della/patch/AccessController.class.patch

E' importante applicare il file patch dalla stessa posizione in cui si trova il file da patchare.

Modifica del link di logout

Per fare in modo che Feng utilizzi il link di logout di Shibboleth è stato modificato il codice HTML dell'applicazione.
Per applicare le stesse modifiche si può utilizzare la patch user_box.patch allegata a questo articolo.

Spostarsi nella directory dove si trova il file in questione e fare un backup dell'originale.

cd /path/di/feng/application/views/application/
cp user_box.php user_box.php.ORIG

 A questo punto applicare la patch al file

patch  -i /dir/della/patch/user_box.patch

ricordo nuovamente l'importanza di rinominare il file con estensione .ORIG e di applicare la patch dalla stessa directory in cui si trova il file.
