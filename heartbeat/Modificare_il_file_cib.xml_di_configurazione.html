2013.10.25 OBSOLETO

 

La configurazione di Heartbeat risiede nel file /var/lib/heartbeat/crm/cib.xml. La modifica di questo file mentre hb è attivo però non avrà l'effetto desiderato, infatti il file viene caricato in memoria e riscritto alla chiusura del programma. Esistono due modi:

  1. spegnere heartbeat, cancellare tutti i file che non sono cib.xml nella stessa directory, modificarlo e riavviare
  2. copiare il file in una directory temporanea, modificarlo, ed a heartbeat acceso eseguire
    cibadmin -R -x /tmp/cib.xml

 Nelle installazioni basate su corosync è possibile modificare la configurazione lanciando il comando:

crm configure edit
