installazione dei pacchetti:

yum -y install qemu-kvm libvirt virt-install bridge-utils virt-install

 

Avviare il servizio libvirt ed abilitarlo per paritre di default:

 

sudo systemctl start libvirtd 
sudo systemctl enable libvirtd

 

Creare l'interfaccia bridge come in esempio esempio :

vim /etc/sysconfig/network-scripts/ifcfg-br0

 

Ed effettuare la configurazione desiderata come da esempio:

DEVICE=br0
TYPE=Bridge
BOOTPROTO=none
ONBOOT=yes
IPADDR=192.168.101.10
NETMASK=255.255.255.0
GATEWAY=192.168.101.1
DNS1=192.168.101.1

 

Adesso procedere alla modifica dell interfaccia ethX:

 

vim /etc/sysconfig/network-scripts/ifcfg-ethx

 

Ed effettuare la configurazione desiderata come da esempio:

NAME=enp0s8
TYPE=Ethernet
ONBOOT=yes
HWADDR=00:1d:60:a2:c1:3c
BRIDGE=kvm01
