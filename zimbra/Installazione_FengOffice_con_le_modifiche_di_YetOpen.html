Effettuare il checkout dal server git di YetOpen

git clone git://mail.ufficyo.com/fengoffice.git /var/www/fengoffice

spostarsi quindi sul branch yetopen

git checkout yetopen

ed eseguire i passi contenuti nel README

 

Una volta installato il sistema, andare nell'Amministrazione alla voce Configurazione > Moduli. Disabilitare quindi i moduli non utilizzati (solitamente Note, Contatti e Collegamenti Web).

 

Troubleshooting

Errore durante il salvataggio di un task

Verificare che per l'utente sia stato impostato il long_timezone dalla scheda delle preferenze.
