Yii2 si basa su composer, il package manager di PHP. Prima di tutto installiamo quello:

curl -sS https://getcomposer.org/installer | php 
sudo mv composer.phar /usr/local/bin/composer.phar
composer.phar global require "fxp/composer-asset-plugin:1.0.0"

 

Quando andiamo a sviluppare qualcosa per Yii2, a meno di un sito dedicato/specializzato faremo sempre un'estensione, in modo che sia facilmente integrabile con altri applicativi sviluppati.

 

Questa guida suppone che i progetti di sviluppo siano in ~/work. Chiameremo la nuova estensione progetto0.

 

Come prima cosa creiamo la directory della nuova estensione e lanciamo il wizard di composer:

mkdir ~/work/progetto0 ; cd $_
composer.phar init

Le risposte da dare:

  1. Package name: yetopen/progetto0
  2. Description: Progetto di prova per l'utilizzo di composer
  3. Author: (indicare i propri dati)
  4. Minimum stability: (lasciare vuoto)
  5. Package type: (lasciare vuoto)
  6. License: (lasciare vuoto)
