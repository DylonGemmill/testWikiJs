La Global Address List (GAL) è l'elenco degli utenti del dominio Zimbra. Viene usata principalmente per l'autocompletamento, se abilitato nelle preferenze.

Come detto la GAL è legata al dominio. Esistono delle installazioni dove vengono gestiti diversi domini interni con un server unico, per cui la GAL deve essere unica per tutti. In questi casi lanciare questi comandi:

zmprov mcf zimbraGalInternalSearchBase ROOT
zmprov mcf zimbraGalSyncInternalSearchBase ROOT
