Zabbix supporta un solo metodo di autenticazione, quindi configurato LDAP non si potrà più usare l'autenticazione interna.

Anche con l'autenticazione di LDAP gli utenti vanno comunque creati in Zabbix, questo per la gestione dei permessi. Creare un utente che esista in Zimbra, ed assegnargli i permessi di amministratore di Zabbix.

Andare in Administration > Authentication e selezionare LDAP dalla drop-down.

  • LDAP host: hostname server zimbra
  • Port: 389
  • Base DN: dc=DOMINIO,dc=IT
  • Search attribute: uid
  • Bind DN: uid=zimbra,cn=admins,cn=zimbra
  • Bind password: password ldap di Zimbra
  • LDAP authentication enabled: SI
