Questo documento si basa sul PDF Zimbra Connector for BlackBerry Enterprise Server Administrator?’s Guide versione 6.5.6, disponibile nell'area download ZCB di Zimbra.com.

  • Creare un account di amministrazione globale sul server Zimbra (ad esempio besadmin@dominio.it), per l'amministrazione dei BlackBerry. Questo account non è strettamente necessario, ma consente di monitorare l'utilizzazione del BES e la generazione di statistiche relative
  • Installare Office 2007 (attenzione, solo questa versione è supportata! Nell'acquisto delle licenze Office 2010 è solitamente incluso il downgrade a 2007), ed assicurarsi che venga aggiornato alla versione SP2
  • Verificare che sia installato l'aggiornamento delle TimeZone, reperibile su http://support.microsoft.com/kb/981793
  • Installare il connettore Zimbra per Blackberry, reperibile nell'area download ZCB di Zimbra.com 
  • Andare quindi in Pannello di controllo > Posta elettronica > Mostra profili 
  • Premere Aggiungi, e come nome dare BlackBerryServer_MailStoreProfile. Premere OK, alla richiesta dei dati dell'account mettere la spunta su Configura manualmente le impostazioni... quindi premere Avanti. Come tipo di server selezionare Altro e quindi Zimbra Collaboration Server. Impostare quindi i seguenti parametri:
    Nome server: hostname.zimbra.com:7071 (impostare il nome host del mailbox server, compreso di porta lato admin)
    Usa connessione sicura: SI
    Indirizzo email: besadmin@dominio.it (l'account amministrativo del BES creato in precedenza)
    Password: *** (la password dell'account besadmin)
  • Sempre dalla funzione Mostra profili aggiungere un nuovo account identico al precedente, con il nome di BlackBerryServer
  • Installare la versione standalone di Microsoft® Collaboration Data Objects 1.2.1 (CDO) dal sito microsoft

Siamo pronti per installare il BES.

Lanciare l'installazione, inserire i dati di intestazione della società ed accettare la licenza.

Selezionare l'installazione di un Blackberry Server.
Ignorare l'avviso della versione di MDAC non corrispondente, immettere la password di Administrator ed il nome del server (hostname?) quando richiesto e procedere con l'installazione.
