Per prima cosa assicurarsi che sul sistema sia installato il supporto per ldap di php (su sistemi debian il pacchetto di riferimento si chiama php5-ldap).

Spostiamoci nella directory config presente nella root dell'applicazione, nel nostro caso /var/www/feng/config. All'interno troviamo vari file, tra cui ldap.config.example.php; copiamolo come ldap.config.php

Apriamo il file appena creato e compiliamo le opzioni come segue:

 

    $config_ldap = array (
        'binddn'    => 'uid=zimbra,cn=admins,cn=zimbra',
        'bindpw'    => 'PASSWORD',
        'basedn'    => 'ou=people,dc=yetopen,dc=it',
        'host'      => '10.22.22.190',
        'uid'       => 'uid'
    );
    return true;

 

dove 10.22.22.190 è il server Zimbra.

ATTENZIONE: in FengOffice il login LDAP non prende tutti i dati dal server, occorre creare gli utenti a mano con gli stessi username presenti su LDAP. Se le impostazioni del file ldap.config.php saranno corrette, qualsiasi password presente in Feng verrà ignorata e verrà utilizzata quella presente in LDAP.
