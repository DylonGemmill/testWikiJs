Per disabilitare la valigetta ad un COS intero

zmprov mc default zimbraFeatureBriefcasesEnabled FALSE

 

Per ottenere tutte le impostazioni del COS

zmprov gc default

 

Per modificare un'impostazione agire come per la valigetta, è possibile verificare anche una sola impostazione specificandola nella chiamata getCos(gc) in questo modo

zmprov gc default zimbraFeatureBriefcasesEnabled
