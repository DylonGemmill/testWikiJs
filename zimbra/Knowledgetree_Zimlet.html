In allegato la zimlet per l'integrazione con Knowledgetree. La zimlet permette di caricare gli allegati delle mail e le mail stesse su Knowledgetree.

Caricamento di un allegato

Vicino ai link per scaricare, rimuovere o mettere in valigetta l'allegato, troviamo anche il link per salvare su Knowledgetree (save to knowledgetree). Cliccando sul link esce una popup che permette di impostare le diverse proprietà di Knowledgetree (autore, tipo file, ecc...). Una volta compilata la form, confermare con il pulsante OK. La zimlet caricherà il documento nella cartella DroppedDocuments/nomeutente di Knowledgetree.

Caricamento di una mail

Trascinare la mail desiderata sull'icona della zimlet nell'elenco delle stesse. Comparirà lo stessa form con i dati aggiuntivi descritta precedentemente.

Installazione

La zimlet in allegato ha il path di Knowledgetree impostato a https://kt.ufficyo.com/knowledgetree

Per modificarlo decomprimere la zimlet in una directory a piacere, modificare il path contenuto nel file config_template.xml e ricreare lo zip della zimlet.

ATTENZIONE:

 - lo zip deve avere il nome della zimlet, ovvero net_oa_kt.zip

 - l'archivio deve contenere direttamente i file della zimlet, e non la directory con i file
