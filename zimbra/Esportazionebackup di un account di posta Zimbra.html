E' possibile salvare il contenuto di un account Zimbra esportando tutto come file .tgz. 

Ogni elemento (Contatti, Calendario...) è esportabile singolarmente, conformati compatibili con altri software, oppure è possibile esportare l'account in blocco, via WebUI o CLI:

  • Preferenze > Importa / Esporta
    Tipo: account
  • zmmailbox -z -m nomeaccount  getRestURL "//?fmt=tgz"  > /tmp/salvataggio.tgz 

Per ripristinare usare il comando

zmmailbox -z -m userinboxsubunder@domain.com postRestURL "//?fmt=tgz&resolve=replace" /tmp/userinboxsubunder.tgz

Questa operazione non esporta però le preferenze ed i filtri. Per i filtri è possibile usare questa procedura. Sul server di origine

zmmailbox -z -m user@domain.com gfrl > /tmp/filters.txt

su quello di destinazione

cat /tmp/filters.txt | sed 's/^/afrl /' | zmmailbox -z -m user@domain.com
