PureFTPd (come tutti gli FTP) ha un limite sul numero massimo di file visualizzaibili in una cartella (2000 predefinito).

Se si vuole aumentare questo limite ad esempio a 5000:

echo "5000 500" > /etc/pure-ftpd/conf/LimitRecursion

(non so cosa sia il secondo valore, noi abbiamo provato con 50 e funziona comunque).

Fatto questo riavviare il demone del server FTP.
