Per aggiungere librerie esterne (sotto forma di file jar, come per esempio ksoap2) procedere come segue:

* click destro sull'icona del progetto nel Package Explorer (sulla sinistra) quindi Properties

* selezionare Java Build Path 

* dalla scheda Libraries selezionare la voce Add External JARs

* sfogliare le cartelle di sistema fino a raggiungere il file JAR e confermare

* selezionare la scheda Order and Export 

* selezionare la libreria JAR appena importata

* confermare con OK

 

Se al lancio del progetto otteniamo l'errore noClassDefFound, chiudere e riaprire Eclipse.
