E' possibile via cli forzare il cambio password. Il comando è:

 

zmprov ma account@domain.com zimbraPasswordMustChange TRUE

 

da eseguire come utente zimbra (su - zimbra).

E' inoltre possibile inviare una mail di notifica agli utenti alcuni giorni prima della scadenza, mediante uno script da schedulare con cron (script in allegato).

Per cambiare la password via cli, invece, usare il comando

zmprov sp user@domain.com nuovapassword
