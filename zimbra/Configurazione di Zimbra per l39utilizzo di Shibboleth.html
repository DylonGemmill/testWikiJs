Per permettere a Zimbra di autenticarsi attraverso Shibboleth, sfrutteremo il servizio di pre-autenticazione messo a disposizione da Zimbra stesso. Modificare come indicato nel file stesso, lo script PHP per il preauth, allegato a questo documento e caricarlo in una locazione accessibile dall'esterno (non è importante dove venga messo, l'importante è che sia accessibile pubblicamente e all'interno della rete di Shibboleth). Dopo di che dare i seguenti comandi per istruire Zimbra ad utilizzare lo script di preauth per il login e l'url di logout di Shibboleth per terminare la sessione.

zmprov mcf zimbraWebClientLoginURL http://zimbra.domain.tld/preauth-url.php
zmprov mcf zimbraWebClientLogoutURL http://zimbra.domain.tld/Shibboleth.sso/Logout

L'ultima versione dello script si trova sempre su git, git+ssh://gitosis@mail.ufficyo.com/web_sso_plugins.git

 

Mettere quindi il file sotto l'autenticazione di Shibboleth aggiungendo alla configurazione di apache il seguente codice

<Location /zimbra_preauth.php>
        AuthType shibboleth
        ShibRequireSession On
        require valid-user
</Location>
