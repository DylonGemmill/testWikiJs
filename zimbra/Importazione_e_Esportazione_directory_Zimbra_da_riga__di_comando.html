Esportazione e importazione cartelle di zimbra

Questa operazione più essere svolta comodamente dal menù preferenze dell'account di Zimbra, ma a volte si può avere l'esigenza di farlo da riga di comando.

Esportazione di un file compresso della cartella Trash: 

zmmailbox -z -m stefano.moreschi@mandolini.it -t 0 getRestURL '//Trash?fmt=tgz' > /tmp/account_cestino.tgz

Visualizzazione delle cartelle dell'account:

zmmailbox -z -m stefano.moreschi@mandolini.it gaf

Importazione del file compresso nell'account restorato:

zmmailbox -z -m restored1118_stefano.moreschi@mandolini.it -t 0 postRestURL '//00CESTINO_RECOVERY?fmt=tgz&resolve=skip' /tmp/account_cestino.tgz

La cartella di destinazione 00CESTINO_RECOVERY va creata prima altrimenti il comando va in errore.
