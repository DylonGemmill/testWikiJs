Scaricare l'installer (install.bin) per Linux amd64 con Java da qui.

Andare nella shell di zimbra ed eseguire:

zmprov gdpak nomedominio.it

che genererà un token di preauth, da memorizzare

 

Quindi eseguire

bash install -i console

Vanno bene tutte le risposte predefinite.

 

Installare la zimlet prelevando il file da /opt/Sheetster_Zimlet/sheetster_zimlet.zip rinominandolo in com_docsfree.zip.
