Specialmente in fase di migrazione è necessario usare una GAL (Global Address List) esterna, che punta verso un server Exchange. Per configurare una GAL verso Exchange è necessario avere delle credenziali di accesso.

La GAL è relativa al dominio, quindi per modificarla collegarsi all'interfaccia web di amministrazione, quindi andare in Dominio > miodominio.it e premere in alto Configura GAL.

  • Modalità GAL: Esterna
  • Tipo di server: LDAP
  • Nome server: ip o hostname del server, porta 3268
  • Filtro LDAP:
    (&(|(displayName=*%s*)(cn=*%s*)(sn=*%s*)(givenName=*%s*)(mail=*%s*))(!(msExchHideFromAddressLists=TRUE))(|(&(objectCategory=person)(objectClass=user)(!(homeMDB=*))(!(msExchHomeServerName=*)))(&(objectCategory=person)(objectClass=user)(|(homeMDB=*)(msExchHomeServerName=*)))(&(objectCategory=person)(objectClass=contact))(objectCategory=group)(objectCategory=publicFolder)(objectCategory=msExchDynamicDistributionList)))
  • Filtro autocompletamento:
    (|(cn=%s*)(sn=%s*)(gn=%s*)(mail=%s*))
  • Base di ricerca LDAP:
    dc=dominioexchange,dc=local
Quindi nella schermata successiva:
  • Mettere la spunta per l'uso del binding
  • Come username mettere: utente@dominioexchange.local
  • Password: la password del suddetto account in exchange
