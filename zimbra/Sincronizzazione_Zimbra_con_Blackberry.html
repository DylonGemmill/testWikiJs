Per sincronizzare i dispositivi BlackBerry con Zimbra occorre:

  • Zimbra Network Edition Standard 
  • ZCB Zimbra Connector for BlackBerry
  • Una licenza per Outlook (Office) 2007 o superiore
  • Un server Windows 
  • Il BlackBerry Enterprise Server [Express]

Il connettore ZCB si appogia al connettore per Outlook, che nel caso di BlackBerry funziona anche sulla standard edition, non professional.

In alternativa esistono delle app per BB che implementano la connessione a server ActiveSync:
