Il supporto diretto di Zimbra a GPG sembra latitare (#6158), nonostante le innumerevoli richieste.

E' possibile comunque criptare le mail in transito usando un'estensione di Firefox, FireGPG. Ufficialmente non è più sviluppata, ma ne è stato fatto un fork che viene mantenuto su github.

Per installarla su linux:

mkdir ~/work
git clone git://github.com/bit/firegpg.git ~/work/firegpg
echo "~/work/firegpg" > ~/.mozilla/firefox/[DIRPROFILO]/extensions/firegpg@firegpg.team

quindi riavviare firefox, ed il wizard di FireGPG dovrebbe avviarsi.
Ovviamente prima è necessario aver impostato GPG sul sistema in uso.

Nella directory di FireGPG è presente anche uno script per generare il file xpi, per eseguire l'installazione tradizionale in firefox.

Attenzione: questa soluzione consente di criptare il testo, ma non gli allegati presenti nell'email!
