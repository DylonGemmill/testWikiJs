Abilitare l'autenticazione LDAP quindi andare in Admin ed inserire i seguenti parametri:

  • host: ip server zimbra
  • port: 389
  • name: uid=zimbra,cn=admins,cn=zimbra
  • pass: password ldap zimbra
  • base: ou=people,dc=yetopen,dc=it (sostituendo yetopen e it con il dominio)
  • filter: uid=%uid
Al momento della prova per funzionare ho dovuto commentare la riga 68 (ldap_start_tls) del file apps/user_ldap/user_ldap.php
