A volte per intercettare degli errori degli utenti, occorre capire a fondo il loro modus operandi.

A questo scopo può essere utile registrare lo schermo e la tastiera del computer in modo da poterli analizzare successivamente.

Per registrare lo schermo è possibilie utilizzare recordmydesktop, utility presente nei repo di molte distro, nel nostro caso serve su ubuntu quindi è possibile installarla con

apt-get install recordmydesktop

Per lanciarla da remoto in modo che salvi direttamente l'output ad una qualità abbastanza bassa e senza audio, possiamo utilizzare il comando

recordmydesktop --on-the-fly-encoding --v_quality 10 --no-sound

In questo caso non specifichiamo un file di output, quindi il video finale sarà memorizzato in un file out.ogv nella directory corrente.

 

Per loggare invece la tastiera possiamo utilizzare il programma logkeys, anche questo presente nei repository di ubuntu, ed installabile quindi con il comando

apt-get install logkeys

Il programma richiede di essere eseguito da root. Per lanciare una registrazione

logkeys --start -o ouput.txt --us-keymap

E' obbligatorio specificare una keymap, altrimenti il programma non funzionerà.

Dopo averlo lanciato il programma rimane in background liberando quindi la console. Per fermare il logger

logkeys --kill

Per loggare la tastiera italiana andare al link

https://code.google.com/p/logkeys/wiki/Keymaps

e scaricare la mappa italiana (file it.map)

quindi lanciare il comando specificando la tastiera con il path del file in questo modo:

logkeys --start -o ouput.txt -m /path/it.map
