In Endian 2.4.1 c'è un bug per cui l'aggiornamento DynDNS non funziona.

Aprire il file /usr/local/bin/setddns.pl e modificare intorno alla riga 387:

if ($settings{'SERVICE'} = 'zoneedit') 
  {$serverdir = '-s'; $serveradd = 'dynamic.zoneedit.com';}
else 
  {$serverdir = ''; $serveradd = '';}

con

if ($settings{'SERVICE'} eq 'zoneedit') 
  {$serverdir = '-s'; $serveradd = 'dynamic.zoneedit.com';}
else 
  {$serverdir = ''; $serveradd = '';}

 

(sostituire l'= con eq nella prima riga dell'if).

 

rif: http://bugs.endian.com/view.php?id=3307
