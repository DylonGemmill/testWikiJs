Rispetto all'uso di PPTP, IPSec risulta più sicuro.
  1. VPN > IPSec: abilitare IPSec;
  2. VPN > IPSec > Mobile clients: abilitare la spunta;
     Abilitare Provide a virtual IP address to clients ed inserire una rete privata non occupata sul firewall;
  3. confermata la scelta premere il pulsante che compare nella barra di stato Create Phase1:
    * Authentication method: Mutual PSK + Xauth;
    * Negotiation mode: aggressive;
    * My identifier: My IP address;
    * Peer identifier: User Distinguished Name, vpnusers@dominio.com;
    * Pre-Shared Key: PSK001 [CAMBIARE!!!] 
    * Proposal checking: Obey;
    * Enctyption Algorithm: AES 128;
    * Hash Algorithm: SHA1;
    * DH Key Group: 2;
    * Lifetime: 86400;
    quindi Salvare e premere il + per visualizzare la Phase2 e premere il pulsante di aggiunta;
  4. * Mode: tunnel;
    * Local Network: LAN;
    * Protocol: ESP;
    * Encryption Algorithms: AES 128 (solo questo!) 
    * Hash Algorithms: SHA1 (solo questo!)
    * PFS key group: off;
    * Lifetime: 28800;
    Salvare ed applicare le modifiche;
  5. Firewall > Rules > IPSec: creare una nuova regola con sorgente, destinazione e protocollo any, a meno di voler imporre restrizioni di accesso;
  6. andare in System > User Manager > Groups: creare un gruppo, salvare, quindi modificare il gruppo ed aggiungere il permesso User - VPN - IPsec xauth Dialin; 
  7. Andare in System > User Manager, aggiungere un utente mettendo come password e come IPsec Pre-Shared Key la password per la VPN.

Configurazione iOS (iPhone, iPad)

Impostazioni > Generali > Rete > VPN: aggiungi connessione VPN, tipo IPSec:
  • server: ip/hostname del server;
  • account: nome utente creato;
  • password: password utente;
  • nome gruppo: vpnusers@dominio.com; 
  • secret: PSK001 (CAMBIARE).
