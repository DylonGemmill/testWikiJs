Inserire nelle impostazioni del server sotto la voce advanced configuration:

ifconfig-pool-persist /root/ipp.txt
Nel file /root/ipp.txt specificare
username,IP
l'username è quello di accesso OpenVPN, l'IP è uno della rete OpenVPN.
 
ATTENZIONE che openvpn assegna una /30, significa che per ogni client si può usare un indirizzo ogni 4. Ovvero se al primo client connesso viene assegnato l'IP .6, il successivo potrà avere .10, quello dopo .14 e così via...
