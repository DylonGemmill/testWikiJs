Client Windows

Scaricare il client ufficiale al seguente link: 

https://openvpn.net/community-downloads/

Cliccare sul file scaricato con il tasto destro e seleziona "esegui come amministratore con e seguire i passi dell'installazione, infine confermare il messaggio di sicurezza di Windows.

Usare lo zip fornito da Yetopen ed estrarlo in C:/Program Files/OpenVPN/config/, modificare i parametri dell' unico file .txt con le credenziali di accesso alla VPN.

Una volta copiata la configurazione, cliccare con il tasto destro sull'icona desktop Openvpn, cliccare su impostazioni e selezionare il tab Compatibilità, e mettere la spunta su Esegui questo programma come amministratore.

Far partire Openvpn con privilegi di amministrazione e cliccare con il tasto destro sull'icona in basso a destra, si visualizzeranno le VPN disponibili configurate, infine per avviare la VPN premere su Connetti.

Se tutto avviene in modo corretto verra visualizzata la notifica di avvenuta connessione.

Client Ubuntu

Aprire il terminale e digitare:

apt-get install openvpn

Copiare lo zip fornito da Yeopen in /etc/openvpn/ e decomprimerlo all'interno:

sudo cp vpn.zip /etc/openvpn/
cd /etc/openvpn/
unzip vpn.zip

Modificare il file .txt inserendo le credenziali in possesso:

vim pass.txt
UTENTE
PASSOWRD

Chiudere il file con :wq

L' ultima configurazione serve per non far partire la vpn in automatico quindi :

vim  /etc/default/openvpn
AUTOSTART="none"
