Con LibreOffice Calc è possibile aprire file CSV da inviare a GLS e modificarli per aggiungere o correggere determinate voci.

Il tracciato impostato sul sito di GLS prevede l'utilizzo del carattere | (pipe) come separatore (che si ottiene premendo Maiusc + \, da non confondere con l che è la L minuscola).

Se si necessita quindi di utilizzare LibreOffice Calc per modificare il file, la procedura da seguire sarà la seguente:

  • aprire il file CSV con LibreOffice
  • nella finestra che compare, sezione Opzioni di sillabazione, selezionare Altri e specificare il carattere | (pipe); cancellare le altre opzioni e confermare col pulsante OK
     
  • effettuare le modifiche che ci interessano
  • al termine delle modifiche selezionare File > Salva con nome
  • nella schermata che ci si presenta selezionare l'opzione Modifica impostazioni filtro e confermare col pulsante Salva

  • un'ulteriore finestra ci chiederà se siamo sicuri di voler mantenere il formato, selezioniamo Usa il formato di Testo CSV

  • nella finestra successiva impostare come Separatore di campo il carattere | (pipe), lasciare in bianco il campo del Separatore di testo
     
  • confermare con il pulsante OK

Avremo quindi il file CSV modificato e pronto per essere caricato su GLS.
