Scaricare il pacchetto di installazione HTML5+PHP dal sito netbeans.org.

Per gli standard PSR-1 e -2 installare il plugin presente su GitHub.

 

Si può creare un progetto da un sorgente esistente. Creare la base del progetto Yii quindi importare il sorgente in Netbeans. Per integrare Yii

  • aprire File > Project properties > PHP Include Path ed aggiungere il path alla root dei Yii;
  • aprire Tools > Options > Miscellaneous > Files nel campo Files ignored by IDE aggiungere alla regexp yiilite\.php. La stringa risultante sarà  "^(yiilite\.php|CVS|SCCS|....".
Riavviare Netbeans.
Per l'autocompletamento usare CTRL+Spazio.
Per l'elenco dei parametri di una funzione: CTRL+p.
