Se tentando di inviare email da Outlook si presenta il messaggio in allegato, occorre seguire le istruzioni che ci  vengono poste per riparare il file danneggiato.

Il programma SCANPST.EXE, nel nostro caso, si trova in C:\Programmi\Microsoft Office\Office12. Il percorso potrebbe variare a seconda delle installazioni. Il programma sembra essere presente solo nelle versioni recenti di Outlook, e non in quelle più datate (Outlook Express).

Per maggior sicurezza è consigliato fare una copia di backup del file Outlook.pst

Una volta avviato, il programma chiede dove si trova il file da analizzare. Dopo una prima analisi bisogna avviare la correzione degli errori. Il programma permette inoltre di effettuare un backup prima della correzione. Se il file Outlook.bak è già presente viene chiesto di sovrascriverlo: è consigliato modificare il path del file di backup, o in alternativa rinominare il file Outlook.bak già presente prima di procedere.
