In Hylafax 6.0.5 di ubuntu c'è un problema con il conteggio delle pagine per alcuni PDF. Ci è stata mandata una patch dallo sviluppatore da HF, e nel mentre che viene accolta da Ubuntu il pacchetto hylafax-client va ricompilato ogni volta che c'è un aggiornamento.

 

Per compilare Hyafax:

cd /usr/src
apt-get build-dep hylafax-client
apt-get source hylafax-client
apt-get install build-essential fakeroot dpkg-dev
cd hylafax-*
patch libhylafax/SendFaxClient.c++ < ../conta_pagine.patch
fakeroot debian/rules binary
dpkg -i ../hylafax-client*.deb
