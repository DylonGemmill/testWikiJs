Questo manuale racchiude le principali funzioni e la loro configurazione.

 

Gestire alcuni prodotti che non sono spedibili

Almeno nella fase iniziale si è deciso che alcuni prodotti troppo ingombranti non verranno spediti. E' possibile limitare lo spedizioniere in diversi modi:

  • impostando i limiti di dimensione nel corriere, ed impostando le dimensioni dell'oggetto nella scheda prodotto (tab Spedizione);
  • impostando una restrizione sul corriere nella scheda prodotto (tab Spedizione). In questo modo è possibile forzare l'uso di uno specifico metodo di spedizione per un determinato prodotto, senza passare per le regole di spedizione.

Aggiunta lingue

Andare nel backend > Localizzazioni > Traduzione > Import/export traduzioni. Selezionare la lingua e premere il pulsante Aggiungere o aggiornare la lingua.

 

Informazioni aggiuntive nel carrello 

Di default è abilitata la visualizzazione di un pulsante PayPal sotto il riepilogo del carrello. E' un modulo installato nella posizione displayShoppingCart (Moduli > Posizioni), rimosso quello non viene più visualizzato. 

 

Modifica campi di registrazione

La modifica dei campi visualizzati va fatta direttamente nel template. I file da modificare sono themes/pt_marris/authentication.tpl, identity.tpl, order-opc-new-account.tpl (quest'ultimo solo se si utilizza one-page-checkout).

 

Backup

Non esiste una soluzione completa ed automatica di backup di file e database insieme.

Il salvataggio del DB si può fare da Parametri Avanzati > Backup DB oppure da PhpMyAdmin, avendo cura poi di scaricare e salvare il backup da qualche parte.

Il salvataggio dei file si può fare tramite FTP.

In caso di server dedicato si possono prevedere operazioni schedulate automaticamente.
