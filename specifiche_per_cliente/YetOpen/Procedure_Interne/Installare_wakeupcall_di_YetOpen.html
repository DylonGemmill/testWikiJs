Accedere al centralino come utente root.

Sbloccare la flash con il comando /usr/lib/mercurius/sbin/flash-open

Rinominare i file /etc/asterisk/extensions.conf e /usr/lib/mercurius/functions.d/wakeupcall.conf in modo da mantenere un copia degli originali.

Aggiungere quindi negli stessi percorsi i file di yetopen.

Modificare il file /etc/asterisk/extensions.conf alla riga 14: impostare la variabile ADM con l'interno di amministrazione (quello che può cancellare le sveglie, nel file è 11).

Modificare il file /etc/asterisk/extensions.conf alle righe da 78 a 84: impostare l'exten a seconda del namespace degli interni (es. è impostato _23601XX, le due X alla fine indicano che copre tutti gli interni di due cifre. Se gli interni sono da 3 cifre, impostare _23601XXX, ecc...)

Bloccare nuovamente la flash con il comando /usr/lib/mercurius/sbin/flash-close

Riavviare asterisk (/etc/init.d/asterisk restart).
