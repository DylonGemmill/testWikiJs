Installare Wacom Sign Pro sulla macchina.

Impostare Firefox per aprire i file PDF direttamente con Wacom:

  • aprire le preferenze di Firefox
  • applicazioni
  • trovare tutte le voci che si riferiscono ai PDF ed impostarle per aprire sempre il file con il programma Wacom

Impostare Firefox in modo che possa aprire i file locali aggiungendo alle impostazioni di configuazione i seguenti valori:

chiave: capability.policy.policynames
valore: localfilelinks

chiave: capability.policy.localfilelinks.sites
valore: http://permitteddomain.com

chiave: capability.policy.localfilelinks.checkloaduri.enabled
valore: allAccess

Per aggiungere i valori:

  1. aprire Firefox e digitare nella barra degli indirizzi about:config, un messaggio ci raccomanda attenzione dato che avremo accesso a tutte le impostazioni del browser
  2. nella lista delle impostazioni che compare, cliccare con il tasto e selezionare Nuovo > Stringa
  3. compare una finestra che chiede il nome del parametri, inseriamo capability.policy.policynames e confermiamo
  4. compare un'altra finestra che chiede un altro valore, inseriamo localfilelinks e confermiamo
  5. ripetere i punti 2, 3 e 4 con le altre coppie chiave/valore specificate prima

Note

Al posto di http://permitteddomain.com bisogna inserire il dominio da cui si vuole permettere l'apertura dei link locali. Possono essere specificati più domin, basta separarli con uno spazio.

E' possibile linkare anche un file di rete con la sintassi file://///SERVER/condivisione/file.pdf gli slash all'inizio sono 5, i primi 3 fanno parte del prefisso che indica a Firefox che si tratta di un file (file:///) gli ultimi 2 sono i primi del percorso di rete (//SERVER/condivisione/file.pdf). Notare inoltre che per il percorso del file si possono utilizzare sia / che \ come in Win, quindi anche un path di questo tipo funziona: file:///\\SERVER\condivisione\file.pdf
