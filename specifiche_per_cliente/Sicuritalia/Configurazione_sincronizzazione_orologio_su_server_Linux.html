Per vedere i dettagli e le impostazioni dell'orologio di sistema usare il comando (funziona solo da Ubuntu 14 in su):

timedatectl status

L'output del comando:

 Local time: Mon 2019-07-01 12:08:57 CEST
 Universal time: Mon 2019-07-01 10:08:57 UTC
 RTC time: Mon 2019-07-01 10:09:02
 Time zone: Europe/Rome (CEST, +0200)
 NTP enabled: n/a
NTP synchronized: no
 RTC in local TZ: no
 DST active: yes
 Last DST change: DST began at
 Sun 2019-03-31 01:59:59 CET
 Sun 2019-03-31 03:00:00 CEST
 Next DST change: DST ends (the clock jumps one hour backwards) at
 Sun 2019-10-27 02:59:59 CEST
 Sun 2019-10-27 02:00:00 CET

Local time è l'ora del server secondo il fuso orario impostato, mentre Universal time è l'ora di Greenwich.

Per impostare il fuso orario

timedatectl set-timezone "Europe/Rome"

Sincronizzazione orologio

>= Ubuntu 16.04

timedatectl set-ntp off
apt -y install ntp
sed -i '/^pool/d' /etc/ntp.conf
echo -e "pool 1.it.pool.ntp.org\npool 0.it.pool.ntp.org" >> /etc/ntp.conf
systemctl restart ntp

< Ubuntu 16

apt-get -y install ntp
sed -i '/^server/d' /etc/ntp.conf
echo -e "server 1.it.pool.ntp.org\nserver 0.it.pool.ntp.org" >> /etc/ntp.conf
/etc/init.d/ntp restart

CentOS

yum -y install ntp
sed -i '/^server/d' /etc/ntp.conf
echo -e "server 1.it.pool.ntp.org\nserver 0.it.pool.ntp.org" >> /etc/ntp.conf
systemctl enable ntpd
systemctl restart ntpd
