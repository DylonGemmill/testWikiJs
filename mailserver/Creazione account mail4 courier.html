Sul  server mail4.waylog.net,per la creazione di un account   è necessario quindi effettuare l'accesso in ssh e svolgere i seguenti passi:

 

Effettuare il login da mysql , accedere al database MAILDB ed effettuare la seguente INSERT sulla tabella USERS(modificare i valori di id,name,maildir,crypt in base alle caratteristiche dell'account): 

 

 

mysql>use maildb;
mysql> INSERT INTO users (id,name,uid,gid,home,maildir,enabled,change_password,clear,crypt) VALUES  ('account@example.it','account@example.it','5000','5000','/var/spool/mail/virtual','example.it/account/','1','1','ChangeMe',ENCRYPT('password account'));

 

 

Se non esistente nella tabella DOMAINS procedere anche alla creazione del dominio(non creare assolutamente il dominio se già presente nella tabella domains) esguendo questa query:

 

INSERT INTO domains (domain,transport,enabled) VALUES ('example.com','virtual:','1');

 

 

una volta creato l'account e il dominio  procediamo alla creazione della directory su filesystem che conterranno le mail del nostro account:

 

mkdir -p /var/spool/mail/virtual/example.com/account/{new,tmp,cur}

 

 

settiamo i giusti permessi:

 

 

chown -R virtual:virtual /var/spool/mail/virtual/example.com/
chmod -R 2700 /var/spool/mail/virtual/example.com/

 

 

Riavviamo postfix:

 

service postfix restart

 

 

Una volta effettua questa operazione, effettuare l'accesso con utente e password impostati in precedenza,
