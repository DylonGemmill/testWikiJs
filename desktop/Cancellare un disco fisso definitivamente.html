Per cancellare definitivamente un file o un'intera partizione ci sono due comandi molto utili in Linux: shred e wipe.

Il primo può essere utilizzato per cancellare un singolo file, ed è installato predefinito su tutte le distribuzioni. Lanciare ad esempio il comando

shred –remove –iterations=50 nome_file.txt

ed il file verrà sovrascritto 50 volte (il default è 25) e poi rimosso.

Per cancellare invece interi dischi o partizioni, si usa il comando wipe.
Avviare il sistema usando una Ubuntu Live, abilitare i repository community (Sistema > Amministrazione > Sorgenti software > repo universe) ed installare il pacchetto wipe.

Quindi, aprire il terminale ed eseguire, ad esempio:

sudo wipe /dev/sda1
