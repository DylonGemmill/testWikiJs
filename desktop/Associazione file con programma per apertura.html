Per modificare l'associazione dei tipi di file con i programmi che devono aprire i file (in Windows XP), aprire una cartella qualsiasi quindi Strumenti > Opzioni carella...

Da qui selezionare la scheda Tipi di file che contiene tutte le associazioni.

 

Nel nostro caso un aggiornamento di Firefox ha marcato i file PDF come HTML Document e li ha associati tutti a Firefox per l'apertura predefinita. Per resettare le impostazioni selezionare il tipo di file PDF e cliccare su Elimina. Quindi cliccare su Nuovo, inserire come estensione pdf e selezionare il programma appropriato per l'apertura (nel nostro caso Adobe Reader).
