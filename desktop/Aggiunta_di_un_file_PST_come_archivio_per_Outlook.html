Può capitare che in Outlook il file PST raggiunga le dimensioni massime consentite, occorre quindi archiviare parte dei messaggi per liberare spazio. Outlook permette di effettuare queste archiviazioni in un altro file PST. A lungo andare anche questo nuovo file arriverà alle dimensioni massime consentite. Per creare un nuovo file PST dove eseguire l'archiviazione occorre procedere come segue:

Creare un nuovo file PST:

  • andare in File > Gestione file dati (o simile a seconda della versione di Outlook)
  • cliccare sul pulsante aggiungi
     
  • inserire i dati necessari (tipicamente un nome ed il percorso del file PST che verrà creato). Il nome verrà utilizzato per identificare il file nell'albero di directory posto sul lato sinistro di Outlook
  • salvare e chiudere la finestra

Nella gestione file dati comparirà quindi il nuovo file, memorizzare il percorso del file PST che servirà in seguito.

Per fare in modo che Outlook utilizzi il nuovo file per l'archiviazione, procedere come segue:

  • selezionare Strumenti > Opzioni
  • nella finestra che si apre, selezionare la scheda Altro
  • nella scheda, selezionare il pulsante Archiviazione Automatica
     
  • nella finestra che si apre, modificare il percorso del file PST alla voce "Sposta i vecchi elementi in" con il percorso del nuovo file PST precedentemente creato
  • confermare col pulsante OK

D'ora in poi Outlook archivierà le email nel nuovo file PST.
