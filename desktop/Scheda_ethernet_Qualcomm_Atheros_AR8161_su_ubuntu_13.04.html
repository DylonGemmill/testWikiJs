La scheda in oggetto non funziona da liveusb con ubuntu 13.04, occorre scaricare i driver e ricaricare il modulo indicato con le seguenti istruzioni:

 

wget -O- https://www.kernel.org/pub/linux/kernel/projects/backports/2013/03/28/compat-drivers-2013-03-28-5-u.tar.gz | tar xz
cd compat-drivers-2013-03-28-5-u/
./scripts/driver-select alx
make
sudo make install
sudo modprobe -r alx && sudo modprobe alx

 

Le istruzioni di cui sopra, funzionano a sorpresa anche da liveusb senza aver installato build-essential
