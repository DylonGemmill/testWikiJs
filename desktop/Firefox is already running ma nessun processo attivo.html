Può capitare che tentando di avviare Firefox dica che c'è già un processo attivo, anche se in realtà task manager o ps non riportino alcun programma in esecuzione.

Questo perché probabilmente sono rimasti appesi i file di lock.

Identificare il percorso del profilo, di solito .mozilla/firefox/[profilename]/ e cancellare i file lock e .parentlock.

Sblocco automatico al login

Creare un file ~/bin/sblocca_firefox.sh con questo contenuto:

#!/bin/bash

rm -f /home/NOMEUTENTE/.mozilla/firefox/NOMEPROFILO/.parentlock
rm -f /home/NOMEUTENTE/.mozilla/firefox/NOMEPROFILO/lock

sostituendo i testi in maiuscolo. Poi creare un altro file ~/.config/autostart/pulisci-firefox.desktop con questo contenuto:

[Desktop Entry]
Type=Application
Exec=bin/pulizia_firefox.sh
Hidden=false
NoDisplay=false
X-GNOME-Autostart-enabled=true
Name[it_IT]=pulizia_firefox
Name=pulizia firefox
Comment[it_IT]=Cancella i file di lock di Firefox
Comment=Cancella i file di lock di Firefox
