Primo accesso

 

Per avviare l'applicazione eseguire il file run nella directory principale. 

La prima volta che apriremo File_Protector dovremo inserire il nome profilo (che è il nome dell'intestatario della carta) e la password (che deve essere uguale al pin della carta), inseriamoli e premiamo OK.

Dopo il primo accesso pr riaccedere basterà selezionare il nome che verrà salvato e inserire il Pin.

 

Firma del documento

 

Una volta eseguito l'accesso sulla finestra che si aprirà clicchiamo su Firma, scegliamo il file che si vuole firmare ed in seguito cliccare su apri.

Cliccare su Aggiungi firma ed in seguito inserire il pin, una volta fatto attendere che la firma venga effettuata.

Cliccare su Salva.

 

 

Sbloccare la carta

 

Se la carta si è bloccata dopo aver sbagliato per 3 volte il Pin cliccare su Dispositivo>Sblocco, bisognerà inserire il Puk e due volte il Pin, infine cliccare su OK.
