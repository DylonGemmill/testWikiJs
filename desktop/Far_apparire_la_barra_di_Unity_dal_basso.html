Rimuoviamo il plugin se abbiamo provato l'installazione manuale con il comando:

rm ~/.compiz-1/plugins/libunityshell.so

Eseguire poi questi passaggi da terminale:

sudo add-apt-repository ppa:paullo612/unityshell-rotated
sudo apt-get update
sudo apt-get install unityshell-rotated libnux-1.0-0 compizconfig-settings-manager

ATTENZIONE: da qui in poi unity potrebbe freezarsi. Se dovesse capitare, entrate in tty0 con CTRL + ALT + F1, loggatevi e riavviate unity con il comando

sudo service lightdm restart

Se la barra è ancora verticale, significa che unity è freezato prima di apportare le modifiche necessarie. Ripetere quindi i passi da qui in poi.

 

Lanciare CompizConfig Settings Manager (comando ccsm) disabilitare la voce Ubuntu Unity Plugin ed abilitare Ubuntu Unity Plugin Rotated. Ubuntu ci segnalerà che il plugin va in conflitto con altre impostazione, selezionare di risolvere il problema e confermare di disabilitare i plugin che creano conflitto. Eseguire logout & login, la barra dovrebbe comparire dal basso!
