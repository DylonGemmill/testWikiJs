Il software A-PDF Content Splitter si occupa di dividere un pdf in pdf multipli sulla base del cambiamento del testo in una detterminata sezione del pdf sulle varie pagine.

La neccessità era quelle di dividere i pdf risultanti in sottocartelle, in base alla parte variabile del nome dei file configurata nelle opzioni delle regole di divisione. Il programma non permette di farlo ma è in grato di eseguire un file .exe alla terminazione della divisione del file.

Creazione del .exe da eseguire:

  • Creare un nuovo file .bat e aprirlo con un editor
  • Il codice dello script che si occupa di prendere tutti i file pdf nella cartella di destinazione dello "splitter" e dividerli in sottocartelle in base al nome è il seguente:
@echo off
rem prendo tutti i file della directory passata come parametro
set "path=C:\Users\liviucalin\DocumentsA-PDF\buste_paga"
for /f "delims=|" %%f in ('dir /b /a-d %path%\*.pdf') do (
	rem per ogni file prendo il nome, dovrebbe essere tra i caratteri _ e .
	for /f "tokens=2 delims=_,." %%A in ("%%f") do (
                rem se la cartella con il nome non esiste la creo
		if not exist "%path%\%%A" mkdir "%path%\%%A"
                rem sposto il file nella cartella corrispondente al nome
		move "%path%\%%f" "%path%\%%A\%%f"
	)
)
  • Una volta salvato per convertire il file da .bat a .exe si può usare un servizio online come ad esempio: http://www.f2ko.de/it/b2e.php

 

Per specificare uno script da esebuire al termine della divisione di un pdf:

  1. Nella sezione "2. Selecting a Rule for Splitting" cliccare l'icona di modifica per modificare una regola o se questa non c'è cliccare l'icona per aggiungere una regola nuova
  2. Andare in "Options" e successivamente nel tab "Trigger for Output Folder"
  3. Attraverso il pulsante "Browse..." selezionare l'applicazione da eseguire precedentemente creata.
