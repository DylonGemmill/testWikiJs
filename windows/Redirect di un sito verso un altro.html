Per effettuare un redirect completo di un sito verso un altro, redirezionando all'url richiesto, creare un file .htaccess con il seguente contenuto:

Options +FollowSymLinks
RewriteEngine on
RewriteRule ^(.*)$ http://nuovo.sito.it/$1 [R=301]

In caso di problemi (internal server error), è possibile commentare la prima riga.
