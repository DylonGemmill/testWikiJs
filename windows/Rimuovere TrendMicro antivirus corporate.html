Il client TrendMicro deve essere installato via server, o richiede una password.

Per rimuovere la richiesta password, andare nel registro di windows al gruppo

HKEY_LOCAL_MACHINE\SOFTWARE\TrendMicro\PC-cillinNTCorp\CurrentVersion\Misc

e impostare la chiave

Allow Uninstall = 1

 

Altrimeti c'è una seconda via, modificando il file Ofcscan.ini, ma non ha funzionato per me.
