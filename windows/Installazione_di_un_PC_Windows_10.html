Scaricare dal sito Microsoft lo strumento di creazione immagine disco Windows 10.
Questo software consente l'upgrade di un PC locale oppure la creazione di supporti esterni di installazione, come USB o DVD.
Per la preparazione del supporto servono 8GB liberi su un computer Windows.

Avviare lo strumento e creare il supporto. Una volta pronto avviare l'installazione senza cavo di rete. In questo modo è possibile creare un account offline, altrimenti l'installazione richiederà obbligatoriamente un account Microsoft online.

Se non diversamente richiesto creare l'account yetopen.

Alla richiesta della password dell'account è possibile lasciare il campo vuoto, in questo modo non vengono richieste le domande di riserva. Sarà possibile impostare una password successivamente.
