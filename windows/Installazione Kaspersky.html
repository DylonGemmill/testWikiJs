Installazione Kaspersky su client Windows

Download del software

Per installare Kaspersky su un client è necessario scaricare il file .exe.

  • Accedere a Kaspersky Business Hub ed accedere all'area di lavoro del cliente sui cui computer si vuole effettuare l'installazione.
  • Accedere alla scheda dei software e scaricarlo

Installazione del software

  • Copiare l'exe sul pc su cui si vuole installare
  • Disinstallare l'eventuale software antivirus già installato sul pc
  • Avviare il programma e seguire le istruzioni a schermo per l'installazione

Una volta installato potrebbe chiedere di riavviare il pc.

La sincronizzazione della licenza avviene potrebbe impiegare parecchi minuti, una volta attivato il prodotto avviare l'aggiornamento di Kaspersky ed effettuare una scansione delle aree critiche, sono 2 operazioni piuttosto veloci.

Se non risulta visibile il client nell'area di lavoro Kaspersky (vedi sezione download software per link) dopo qualche minuto è possibile forzare la sincronizzazione avviando come amministratore cmd ed eseguendo  

C:\Program Files (x86)\Kaspersky Lab\NetworkAgent\klnagchk.exe
