scaricare ed installare sql server express

aprire la configurazione di sql quindi andare in configurazione di rete > protocolli, abilitare il tcp/ip e riavviare il servizio

andare in servizi > sql server browser ed abilitare l'avvio automatico

 

il server risponde ma non è ancora possibile connettersi dall'esterno con l'utente perché è stata selezionata l'autenticazione di windows come unico metodo di autenticazione disponibile

 

abilitare l'autenticazione mista (sql e windows) da sql studio selezionando il proprio server e andando in proprietà > sicurezza, da qui selezionare il metodo di autenticazione "Autentivazione di SQL Server e di Windows"

http://stackoverflow.com/questions/8075784/error-message-provider-shared-memory-provider-error-0-no-process-is-on-th

 

per aggiungere un'utenza da utilizzare per la connessione dall'esterno https://chartio.com/docs/datasources/connections/details/sqluser
