Per fare il deployment dell'applicativo in produzione creare nella directory principale del progetto un file deploy.sh

Nel file inserire il seguente script, sostituento le parti variabile tra parenesi triangolari:

git ls-files -z | rsync $1 --size-only -e "ssh -i <PATH_CHIAVE_PRIVATA> -p<PORTA_SSH>" --stats --files-from - -lrvz0 . <UTENTE>@<HOST>:<PATH_AL_PROGETTO>
rsync $1 --delete --delete-after --exclude .git -e "ssh -i <PATH_CHIAVE_PRIVATA> -p<PORTA_SSH>" --stats -lrvzHt vendor/ <UTENTE>@<HOST>:<PATH_AL_PROGETTO>


Perche lo script funzioni bisogna prima aggiungere la propria chiave pubblica sul server per l'utente

Salvare il file e aggiungergli il permesso di esecuzione: `chmod +x <PATH_LOCALE_AL_PROGETTO/deploy.sh`
