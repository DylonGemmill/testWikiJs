In caso di errore le chiamate SOAP di PHP scatenano delle eccezioni, ma di default non tutti gli errori possono essere intercettati con try/catch e generano errori di PHP che andrebbero gestiti con error_report.

Per gestire tutto localmente con il primo metodo istanziare l'oggetto SOAP con il parametro exception, e mettere una @ davanti alla definizione del servizio per sopprimere eventuali errori:

$soap_options = array(
 'exceptions' => true,
);
try {
 $client = @new SoapClient($url, $soap_options);
} catch (SoapFault $e) {
 error_log("Errore connessione SOAP: ".$e->getMessage());
}
