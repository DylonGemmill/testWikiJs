Come base useremo una Ubuntu 10.04 LTS server.

Andare al sito knowledgetree.org e scaricare l'installer universale (qui allegato). 

Andare su packages.ubuntu.org e scaricare il pacchetto swftools, che da qualche distribuzione non è più incluso. L'ultima versione pacchettizzata è per karmic, disponibile a questo url: swftools per karmic. Scaricato eseguire:

dpkg -i swftools_0.9.0-0ubuntu1_amd64.deb
apt-get -f install

Lo script configurerà i repository di Zend e Knowledgetree, ed installerà le dipendenze necessarie, tra cui Apache2, MySQL e PHP5.

Per loggarsi utilizzare le credenziali
username: admin

password: admin 

 

Come da istruzioni a fine installazione, avviare il demone per avere maggiori prestazioni in ricerca, indicizzazione e creazione dei pdf

# cd /usr/share/knowledgetree-cs
# sudo ./dmsctl.sh start
