Se su Fedora non funziona il webserver e nei log trovate l'errore Class 'DOMDocument' not found, può essere semplicemente che non siano state installate le librerie XML di PHP.

Per installarle

yum install php-xml

successivamente riavviare chi si occupa di interpretare il php (nel mio caso php-fpm)

systemctl restart  php-fpm.service
