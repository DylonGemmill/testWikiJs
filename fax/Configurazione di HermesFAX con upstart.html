Creare il file /etc/init/hermes.conf con il seguente contenuto:

# hermes.conf
#

start on stopped rc RUNLEVEL=[2345]

stop on runlevel [!2345]

exec /var/spool/hylafax/bin/hermescron
respawn

 

quindi /etc/init/hermessend.conf

# hermessend.conf
#

start on stopped rc RUNLEVEL=[2345]

stop on runlevel [!2345]

exec /var/spool/hylafax/bin/hermessendcron
respawn

 

Per avviare i due servizi:

start hermes; start hermessend
