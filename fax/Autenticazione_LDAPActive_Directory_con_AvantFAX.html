Abilitare il modulo di autenticazione apache con i comandi:

a2enmod authnz_ldap
service apache2 restart

 

 

Nella root di avantfax creare il file .htaccess con questo contenuto:

 

AuthName "Inserire utente e password per l'accesso al dominio"
AuthType Basic
AuthBasicProvider ldap
AuthzLDAPAuthoritative off
AuthLDAPURL "ldap://pdc.domain/dc=organization,dc=subdomain,dc=domain?sAMAccountName?sub?(objectClass=*)"
AuthLDAPBindDN "user@domain."
AuthLDAPBindPassword password
require valid-user

 

Sostituendo l'url LDAP con l'ip del server PDC, e AuthLDAPBindDN utente e password con un utente e una password abilitati ad accedere all'elenco degli utenti del dominio.

Attenzione, gli utenti devono comunque essere configurati in AvantFax!

 

Per Zimbra  andranno personalizzati questi parametri:

AuthLDAPURL ldap://yourzimbraserver:389/ou=people,dc=yourdomain,dc=com?uid?sub?(objectClass=organizationalPerson)
AuthLDAPBindDN uid=zimbra,cn=admins,cn=zimbra
AuthLDAPBindPassword yourldappassword

Dove yourldappassword è l'output di zmlocalconfig -s ldap_root_password.
