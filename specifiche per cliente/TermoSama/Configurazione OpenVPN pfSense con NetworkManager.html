Copiare i file yopf.crt e yopf.key presenti in ufficio/VPN/openvpn/ in locale sul PC, quindi andare in NM per configurare una nuova VPN, selezionare come opzione OpenVPN. Impostare questi parametri:

  • Gateway: mail.ufficyo.com
  • Tipo: Password
  • Nome utente: nome utente di UfficYO
  • Password: password di UfficYO
  • Certificato CA: puntare al file yopf.crt copiato in precedenza
Premere Avanzate
  • Generale > Abilitare la compressione LZO
  • Sicurezza > Cifrario: AES-128-CBC (forse non serve)
  • Autenticazione TLS > Corrispondenza per oggetto: ufficyo (forse non serve)
  • Autenticazione TLS > Abilitare Usare autenticazione TLS aggiuntiva 
  • Autenticazione TLS > File chiave: puntare al file yopf.key copiato in precedenza
  • Autenticazione TLS > Direzione della chiave: 1
Nella scheda IPv4, premere il pulsante Instradamenti ed abiltare la spunta Usare questa connessione solo per risorse sulla sua rete. 
Se ci sono problemi strani di comunicazione, come ad esempio caricamento parziale delle pagine oppure errori nel listing delle cartelle samba, nelle opzioni avanzate della VPN attivare:
  • Usare una Maximum Transmission Unit (MTU) personalizzata per il tunnel: 1500
  • Limintare il Maximum Segment Size TCP del tunnel
