1. Eseguire il CD di Telestra
  2. Impostare come stampante predefinita PDFCreator
  3. Esegure da Telestra la stampa di tutte le categorie (Menù File > Catalogo generale (?))
  4. Aprire il PDF generato
  5. Selezionare tutto il testo ed incollarlo in un file di testo
  6. Copiare il file TXT sul server
  7. Pulire le righe non di categoria con il comando 
    grep '^0' categorie.txt >  categorie_clean1.txt
  8. Sostituire il testo ', ' con '";"'
    sed -e 's/,\ /\";\"/g' categorie_clean1.txt > categorie_clean2.txt 
  9. Inserire le virgolette all'inizio ed alla fine di ogni riga
    sed -e 's/^/\"/g' categorie_clean2.txt > categorie_clean3.txt 
    sed -e 's/$/\"/g' categorie_clean3.txt > categorie_clean4.txt
  10. Importare il file da phpmyadmin nella tabella callcenter_categorie
