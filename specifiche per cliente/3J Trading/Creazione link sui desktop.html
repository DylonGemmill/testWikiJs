Affinche i link sul desktop funzionino correttamente bisogna montare la condivisa come segue

Creare la cartella per il mount:

sudo mkdir /CONDIVISA

Aggiungere una riga al file /etc/fstab come segue: 

//<ip_server>/condivisa /CONDIVISA cifs username=<nome_utente>,password=<password_utente>,_netdev 0 0

Aggiungere la cartella /CONDIVISA ai preferiti in modo che possano accedervi comodamente

Con Gnome a quanto pare è complicato creare dei collegamenti sul desktop di file presenti sulla rete.

In particolare Sonia usa molto questi link, e di tanto in tanto ce li chiede.

 

Per creare questi link si può procedere in due modi:

  • collegandosi al server, andando nella home degli utenti, quindi in Scrivania, si può copiare il link come quelli esistenti. Sul server è un dead-link perché la directory a cui punta, condivisa, è un link alla directory .gvfs/ dove viene montata tramite gnome la condivisione del server, quindi va linkato alla cieca;
  • collegandosi sul PC dell'utente, con il login dell'utente, si entra nella cartella e si fa il link, a questo punto tradizionale, seguendo il percorso degli altri (path relativo).
