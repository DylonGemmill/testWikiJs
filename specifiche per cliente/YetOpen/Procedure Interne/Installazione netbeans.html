Per installare netbeans

wget -c http://download.netbeans.org/netbeans/8.2/final/bundles/netbeans-8.2-linux.sh
chmod +x netbeans-8.2-linux.sh
sudo ./netbeans-8.2-linux.sh

Scegliere di personalizzare l'installazione selezionando solo i pacchetti:

  • Base IDE
  • Html5/Javascript
  • PHP
  • Feature on Demand


Troubleshooting

Solitamente non richiede altri pacchetti ma è possibile sia necessario installare libcanberra-gtk
sudo apt-get install libcanberra-gtk-module
