ERRORE UMANO

In caso di errore umano (modifica errata, cancellazione, ecc...) si ripristina dal disco USB con una perdita pari al massimo a 8 ore.

PROBLEMA HARDWARE

In caso di problema hardware si preleva la macchina remota portandola in ufficio con una perdita di dati pari al massimo una giornata di lavoro. Una volta acquistato il nuovo server, si ripristinerà tramite rsync e dd a partire dalla macchina remota.

IMPOSSIBILITA' USO UFFICIO

Si trasporta il server dell'ufficio in un'altra location. La perdita di dati è nulla.
