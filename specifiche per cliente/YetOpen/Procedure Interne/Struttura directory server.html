  • marketing → contiene tutti i documenti con cui abbiamo pubbliciziamo l'azienda, compresi depliant e company profile
  • prodotti-servizi → contiene tutte le informazioni sul nostro portafoglio prodotti, compresi i depliant informativi

Le sottocartelle attualmente non sono schematizzate in quanto a breve sposteremo tutto su KnowledgeTree. In quel momento decideremo una struttura definitiva. Per ora e' importante sapere che il file appunti_cliente.txt.gpg deve contenere tutte le informazioni utili alla gestione dei servizi del cliente, che la sottocartella dettaglihardware contiene un file dettaglihardware (è presente il template in template_bkup) dove vengono memorizzati tutti i dati relativi all'hardware del cliente (ram, cpu, scheda madre, quanti slot ram disponibili, quanti occupati, ecc... con i dai specifici) in modo da poter eventualmente acquistare ricambi o potenziamenti senza doverci muovere dall'ufficio. Sempre in dettaglihardware è presente anche una sottocartella FOTO che contiene le foto degli apparati dei clienti.
