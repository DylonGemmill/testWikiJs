Questa procedura consente di reimpostare la password root di MySQL

  1. Spegnere MySQL
    service mysql stop
  2. Avviare MySQL manualmente facendo omettere il controllo dei privilegi
    mysqld_safe --skip-grant-tables &
  3. Entrare in una shell MySQL
    mysql -u root
  4. Aggiornare la password con la nuova desiderata
    update user set password=PASSWORD("NEW-ROOT-PASSWORD") where User='root';
flush privileges;
quit;
  5. Identificare il processo mysqld figlio di mysqld_safe ed iniviare un segnale di kill
  6. Riavviare MySQL
    service mysql start

Questa procedura funziona su Ubuntu e tutte le distribuzioni che usano upstart. Per le vecchie distro usare i soliti comandi /etc/init.d/mysql, in particolare al punto 5 è possibile usare la stop invece della kill.
