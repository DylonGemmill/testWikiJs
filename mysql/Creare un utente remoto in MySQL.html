L'accesso da remoto a MySQL, aggiungere un utente con dominio %

 

mysql> create user nomeutente@'%' identified by 'password';

Poi nella configurazione di mysql (file /etc/mysql/my.cnf) modificare la proprietà bind-address come segue

 

bind-address 0.0.0.0

 

Con la configurazione sopra riportata viene messo il server in ascolto su qualsiasi interfaccia invece che sulla sola localhost (come era di default). Al posto di % e 0.0.0.0 è possibile effettuare una configurazione più precisa inserendo l'interfaccia del server desiderata (quindi ip del server e non del client).
