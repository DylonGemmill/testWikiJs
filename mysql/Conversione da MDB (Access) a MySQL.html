Come prima cosa installare gli mdb-tools

apt-get install mdbtools

quindi creare lo schema con il comando

mdb-schema database.mdb mysql | grep -v ^DROP | grep -v ^- | mysql -u user_name -p new_mysql_database

ed importare i dati con

mdb-export -I database.mdb table_name | sed -e 's/;/\;/g'| sed -e 's/)$/);/' | mysql -u user_name -p new_mysql_database

 Uso base
mdb-tables elenca le tabelle contenute nel file access
mdb-export invece dato file e nome della tabella stampa il contenuto del file in formato csv (occhio a reinderizzare l'output)
