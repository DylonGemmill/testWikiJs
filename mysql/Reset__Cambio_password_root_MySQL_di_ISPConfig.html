Se si vuole modificare la password di MySQL senza sapere quella esistente:

service mysql stop
mysqld_safe –skip-grant-tables &
mysql -u root mysql

Nella shell di MySQL 

update user set Password=PASSWORD(‘mynewpassword’) WHERE User=’root’;
exit

quindi terminare l'istanza di MySQL lanciata manualmente, lanciando fg e poi CTRL+C.

Riavviare MySQL:

service mysql start

 

Al termine modificare la password usata da ISPConfig per la creazione dei nuovi DB nel file 

/usr/local/ispconfig/server/lib/mysql_clientdb.conf
