Se capita l'errore in oggetto significa che un determinato comando è stato bloccato nella nostra versione di mysql, probabilmente per questioni di sicurezza. Per poterlo comunque utilizzare occorre abilitarlo dal file di configurazione /etc/mysql/my.cnf

Nel mio caso l'errore viene generato dall'istruzione LOAD DATA LOCAL INFILE, perciò occorre sbloccare l'istruzione inserendo la riga

load-infile = 1

 

nella sezione [client] del file di configurazione. Una volta modificato il file, riavviare mysql con

/etc/init.d/mysql restart
