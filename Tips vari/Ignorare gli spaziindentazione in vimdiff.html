Capita di dover gestire delle differenze con vimdiff (aka vim -d) ma il programma evidenzia quasi tutto perché il file è stato preso da un altro programma che ha modificato indentazione e/o spazi bianchi vari.

 

vim permette di ignorare questi spazi una tantum via riga di comando, con la seguente sintassi

vimdiff -c 'set diffopt+=iwhite' file1.php file2.php

 

oppure in modo permanente, inserendo nel proprio file delle preferenze (~/.vimrc) la seguente opzione

set diffopt+=iwhite
