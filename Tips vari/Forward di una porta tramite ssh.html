E' possibile fare il forward di una porta tramite ssh.

Ad esempio se sulla rete del cliente c'è un web server all'IP 192.168.0.2 ma non è pubblicato, posso usare il server dove risiede ssh per fare da ponte.

Sul mio client locale eseguo:

 

ssh -L 0.0.0.0:9030:192.168.0.2:80 root@server-ssh  -N

in questo modo ssh metterà in ascolto la porta locale sul client 9030 che punta al web server del cliente.
