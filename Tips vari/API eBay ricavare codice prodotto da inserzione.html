Per poter ricavare l'intera inserzione di un oggetto eBay dalle apposite API, occorre utilizzare la chiamata GetSingleItem passando (oltre all'itemID) il parametro IncludeSelector settato a Description. Un esempio di XML

<?xml version="1.0" encoding="utf-8"?>
<GetSingleItemRequest xmlns="urn:ebay:apis:eBLBaseComponents">
<ItemID>140745087435</ItemID>
  <IncludeSelector>Description</IncludeSelector>
</GetSingleItemRequest>?

 

Il dato viene poi tornato nella variabile Description dove i caratteri sono però codificati in caso di codice HTML (i.e. &lt; al posto di <).

 

Un esempio di codice PHP per prelevare il codice prodotto

preg_match ('/Product Code: ([^&]+)&/', $s, $m);

Il primo parametro è il pattern di ricerca in sintassi perl. Le parentesi tonde servono per indicare i valori da tornare come match ($m). Il secondo parametro è la stringa in cui viene effettuata la ricerca.
