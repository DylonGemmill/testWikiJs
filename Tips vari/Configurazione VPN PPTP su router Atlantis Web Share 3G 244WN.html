Aprire l'amministrazione del router, cliccare su Advanced, quindi Configuration > VPN > PPTP.

 

Compilare i campi come da schermata, in dettaglio:

  • Name: nome identificativo della connessione
  • Connection Type: Remote access
  • Type: Dial in
  • Private IP address...: un IP da assegnare al client durante la connessione. deve essere un IP della rete LAN del router
  • Username: nome utente della connessione VPN
  • Password: password vpn
  • Auth type: Chap
  • Data Encryption: Enable
  • Key Length: 128bit
  • Mode: stateless
Quindi premere il pulsante Add, abilitare la spunta sulla colonna Enable poi Save config to flash, quindi Apply.
Per modificare un tunnel VPN occorre disabilitare la voce, quindi premere il radiobutton relativo alla riga, modificare i campi e al termine premere Edit/Delete. Idem per cancellare (usando l'aposita colonna).
