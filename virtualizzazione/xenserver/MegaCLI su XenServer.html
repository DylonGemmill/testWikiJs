Installazione

Scaricare il software MegaCLI dal sito lsi.com oppure da workarea/prodotti-e-servizi/xen-citrix/MegaCli.zip

 

Copiare il file sullo XenServer in /tmp ed eseguire

tutto con il comando:

cd /tmp
 unzip MegaCli.zip
rpm -i linux/*.rpm

Una volta installato, verificare la presenza dei file in /opt/MegaRAID.

Copiare lo script di monitoraggio da ufficio/workarea/prodotti-servizi/server/megacli-statusd.sh in /root/bin, ed in /etc/crontab aggiungere:

 

32 */2 * * *  root    /root/bin/megacli-statusd.sh

Verificare in /etc/ssmtp/ssmtp.conf la configurazione (Installazione di nullmailer, per la consegna della posta) ed inviare una mail di test.

 

Comandi utili

Mostra informazioni sulla scheda raid:

/opt/MegaRAID/MegaCli/MegaCli -AdpAllInfo -aAll

Lista di tutti i drive logici:

/opt/MegaRAID/MegaCli/MegaCli -LDInfo -LALL -aAll

Lista di tutte le informazioni fisiche:

/opt/MegaRAID/MegaCli/MegaCli -PDList -aAll
