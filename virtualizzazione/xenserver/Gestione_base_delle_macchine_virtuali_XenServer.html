La gestione delle macchine virtuali di un host XenServer è gestibile tramite XenCenter, il software gratuito per Windows fornito da Citrix. E' liberamente scaricabile da qui:

http://www.citrix.com/lang/English/lp/lp_2307783.asp

(o comunque dal sito citrix cercando xencenter download, o xenserver extra downloads).

 

Prima di usare XenCenter, ricordiamo che è possibile riavviare le macchine virtuali dal sistema operativo stesso, quindi collegandosi in RDP/VNC per windows, o in VNC/ssh per linux.

 

Dopo aver installato XenCenter, premere il pulsante Add new server per configurare il server Xen. Inserito indirizzo e credenziali, il server verrà aggiunto alla lista a sinistra e comparirà anche l'elenco delle macchine virtuali installate, in verde quelle accese ed in rosso quelle spente:

 

Cliccando con il tasto destro su ogni Virtual Machine (VM) compariranno le opzioni ad essa disponibili. In particolare, per quelle spente sarà possibile unicamente accendere, mentre per quelle in esecuzione sarà possibile inviare il comando di spegnimento/riavvio, o alternativamente forzare questi comandi.

ATTENZIONE, forzando lo spegnimento o il riavvio si esegue un'operazione non gestita dal sistema operativo, quindi potrebbe implicare una perdita di dati, o un non corretto salvataggio delle informazioni!

Non usare le altre funzioni se non si è sicuri di quello che si sta facendo!
