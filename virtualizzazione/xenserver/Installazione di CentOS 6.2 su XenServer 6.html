L'installazione tramite template di XS6 è molto semplice, durante la creazione della nuova VM indicare Centos 6.2 64bit come template di origine, verranno chieste veramente pochissime informazioni ed in pochi minuti.

Al primo reboot però occorre effettuare alcune operazioni, in quanto la configurazione è molto basilare, ad esempio non viene nemmeno installata la rete.

Collegarsi in console e modificare il file /etc/sysconfig/network-scripts/ifcfg-eth0. Le prime due righe dovrebbero essere già compilate, le altre vanno modificate come in questo esempio:

DEVICE="eth0"
HWADDR="C2:98:D2:EB:F2:40"
NM_CONTROLLED="yes"
ONBOOT="yes"
NETMASK=255.255.255.0
IPADDR=192.168.5.4
GATEWAY=192.168.5.254
 NETWORK=192.168.5.0
USERCTL=no
DNS1=8.8.4.4
DNS2=4.2.2.2 

in caso si voglia una configurazione con dhcp:

DEVICE=eth0
BOOTPROTO=dhcp
ONBOOT=yes

 

Per impostare l'hostname modificare il file /etc/sysconfig/network, parametro HOSTNAME.

Per rendere effettiva la configurazione della rete riavviare il servizio network. Per rendere effettivo l'hostname occorre riavviare il server.

 

Per abilitare l'accesso ssh tramite chiavi modificare il file /etc/ssh/sshd_config ed abilitare la riga 
PubkeyAuthentication yes

quindi riavviare sshd. 

 

Installare alcuni componenti aggiuntivi:

yum install vim screen

 

Modificare l'ultima riga del file /etc/aliases per fare in modo che le mail destinate a root vadano verso support@yetopen.it. Quindi eseguire il comando newaliases.
