Per inviare mail dal sistema

In XS è già installato SSMTP, un mailer molto semplice che emula sendmail ed invia i messaggi ad un server smtp.

Modificare /etc/ssmtp/ssmtp.conf:
mailhub=10.0.0.1
rewriteDomain=iltalentocoop.it

Invio mail da linea di comando

yum install mailx

E' possibile abilitare i repo standard di CentOS per installare software aggiuntivo. Per fare questo mettere a enabled=1 i repo che interessano in /etc/yum.repos.d/CentOS-Base.repo.?
