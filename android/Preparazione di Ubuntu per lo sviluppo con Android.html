Lo sviluppo di applicazioni Android è agevolato dall'uso di Eclipse.

 

Scaricare e scompattare l'Android SDK in una cartella a piacere.

 

Installiamo Eclipse con

apt-get install eclipse eclipse-pdt

 

quindi aprire Eclipse ed andare in Help > Install new software... .

Da parte al selettore premere Add ed inserire l'url del repo ADT per Eclipse (preso da qui):

https://dl-ssl.google.com/android/eclipse/

selezionare il repo appena aggiunto, quindi mettere la spunta su Developer Tools e premere Next, e nella pagina successiva Finish. Partirà il download in background. Terminato verrà richiesto il riavvio di Eclipse. Quindi andare in Windows > Preferences, sul pannello di sinistra selezionare Android e specificare il path dove è stato estratto l'SDK, e confermare.

Sempre in Windows > Android SDK and AVD Manager andare su Available packages. Espandere Android Repository e spuntare

  • Documentation or Android [...]
  • SDK Platform Android [...]
  • Samples for SDK API [...]

quindi premere Install selected.
