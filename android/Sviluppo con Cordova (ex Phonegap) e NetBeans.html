Installare nodejs usando lo script di nodesource:

curl -sL https://deb.nodesource.com/setup | sudo bash -
apt-get install -y nodejs
npm install -g cordova

 

Scaricare e scompattare sul sistema le SDK Android (pacchetto completo).

In Netbeans andare in Tools > Plugins > Settings ed aggiungere un nuovo repository di nome NBAndroid con url http://nbandroid.org/updates/updates.xml.  

Una volta riavviato NB andare in Tools > Options > Miscellaneous > Mobile Platforms ed inserire la posizione dell'SDK estratto prima (ad esempio /home/maxxer/work/android/adt-bundle-linux-x86_64-20140702/sdk).

 

A questo punto è possibile creare un nuovo progetto, categoria HTML5, tipo Cordova.
