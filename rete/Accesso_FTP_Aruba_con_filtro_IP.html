Se non si riesce ad accedere allo spazio FTP con gli stessi dati con cui si accede al pannello di controllo, il problema può essere dato dal fatto che Aruba filtra gli accessi FTP sulla base di una lista di IP.

Per aggiungere un nuovo IP alla lista occorre loggarsi qui http://admin.aruba.it specificando il dominio relativo (attenzione, il dominio del sito web e non il dominio per l'accesso FTP) ed inserendo gli stessi utente e password utilizzati per l'accesso FTP.

Nel pannello di controllo, nella descrizione della voce Test FTP, c'è il link "Filtra accessi ftp" che porta all'interfaccia di gestione.

Da qui inserire il proprio IP nel campo apposito (se non lo si conosce è specificato in alto vicino al nome utente), aggiungerlo all'elenco con il pulsante AGGIUNGI IP e salvare inserendo il codice captcha e cliccando il tasto SALVA.

 

Se la connessione FTP non dovesse risultare immediatamente funzionante, aspettate qualche minuto e riprovate.
