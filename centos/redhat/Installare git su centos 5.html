Ai tempi di Centos 5 git non era ancora considerato maturo e quindi non è presente nei repository standard della distribuzione. Per poterlo installare senza doverlo compilare, è possibile aggiungere i repository di rpmforge e quindi installarlo con yum in questo modo:

# wget http://packages.sw.be/rpmforge-release/rpmforge-release-0.5.2-2.el5.rf.x86_64.rpm
# rpm --import http://apt.sw.be/RPM-GPG-KEY.dag.txt
# rpm -K rpmforge-release-0.5.2-2.el5.rf.x86_64.rpm
# rpm -i rpmforge-release-0.5.2-2.el5.rf.x86_64.rpm
# yum install git
