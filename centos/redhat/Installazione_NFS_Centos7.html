Installazione dei pacchetti NFS

yum install nfs-utils nfs-utils-lib

 Abilitare i servizi all'avvio:

systemctl enable rpcbind
systemctl enable nfs-server
systemctl enable nfs-lock
systemctl enable nfs-idmap

Effettuare lo start dei serivizi:

systemctl start rpcbind
systemctl start nfs-server
systemctl start nfs-lock
systemctl start nfs-idmap

Configurare il file di exports per condividere il percorso desiderato:

 

vi /etc/exports
/home   *(rw,sync,no_subtree_check,no_root_squash,async)
