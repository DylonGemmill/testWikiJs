Per controllare l'integrità di un disco fisso si possono usare diversi tool. In caso il filesystem sia nativo linux, fsck con l'opzione -f farà una verifica del filesystem.

Altrimenti è possibile fare dei test di lettura/scrittura non invasivi usando il comando badblocks, parte del pacchetto e2fsprogs.

badblocks -n -v /dev/sdb

Se il disco supporta SMART è possibile interrogarne lo stato con il comando smartctl del pacchetto smartmontools:

smartctl -a /dev/HDD_DEVICE

Per abilitare smart sul disco, se disabilitato, lanciare il comando precedente con l'opzione -s al posto di -a. Per lanciare un test etensivo di SMART sul disco, eseguire il comando seguente. Siccome viene eseguito in background, è possibile verificarne lo stato con il secondo comando:

smartctl -t long /dev/HDD_DEVICE
smartctl -l selftest /dev/HDD_DEVICE
