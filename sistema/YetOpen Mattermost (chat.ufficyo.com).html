È installato embedded in GitLab. 

 Il file  di configurazione è in /var/opt/gitlab/mattermost/config.json, i parametri sono descritti qui.

Linea di comando

Il comando da usare è questo, sostituendo version in fondo con il comando da eseguire.

sudo -u mattermost /opt/gitlab/embedded/bin/mattermost --config=/var/opt/gitlab/mattermost/config.json version
