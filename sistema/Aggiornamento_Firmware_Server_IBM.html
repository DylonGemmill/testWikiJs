Le seguenti operazioni vanno eseguite loggandosi come root in ambiente grafico.

 

Scaricare il software UpdateXpress di IBM per aggiornare il sistema (funziona solo su alcuni sistemi operativi, noi usiamo SUSE Linux Enterprise Server 11 Edition):
* andare sul sito ufficiale di IBM http://www.ibm.com/us/en/
* cercare "updatexpress" nel campo di ricerca
* il primo risultato dovrebbe portarvi alla pagina del software da cui poter scaricare l'installer
* selezionare la voce corrispondente al nostro sistema operativo nel PRIMO elenco (nel nostro caso quello per SUSE Linux Enterprise Server 11 Edition, installer file name: ibm_utl_uxspi_9.41_sles11_32-64.bin NON uxspi440.sles11)
* selezionare il metodo HTTP e continuare
* accettare i termini e le condizioni che escono in un popup
* scaricare tutti i file che vengono proposti nella stessa cartella
ibm_utl_uxspi_9.41_anyos_noarch.chg (7.08 KB)
ibm_utl_uxspi_9.41_anyos_noarch.txt (61.47 KB)
ibm_utl_uxspi_9.41_sles11_32-64.bin (113.22 MB)
ibm_utl_uxspi_9.41_sles11_32-64.xml (25.59 KB)
* aggiungere i permessi di esecuzione al file bin (chmod +x ibm_utl_uxspi_9.41_sles11_32-64.bin)
* eseguirlo (./ibm_utl_uxspi_9.41_sles11_32-64.bin)
* se il programma non dovesse partire installare Java dall'indirizzo http://www.java.com/it/download/

Wizard del programma:
* selezionare la lingua italiana ed accettare i termini di licenza
* Welcome - non selezionare la ricerca di aggiornamenti per UpdateXpress e premere Next
* Update Task - selezionare Update the local machine e premere Next
* Update Location - selezionare Check the IBM web site e premere Next
* Update Type - selezionare UpdateXpress System Packs (UXSP's)
* Proxy Settings - non selezionare l'opzione (dipende dalla configurazione della vostra rete)
* Target Directory - come default mette quella in cui è stato lanciato il binario, per noi va bene
* Acquire Updates - l'applicazione scarica gli aggiornamenti dal sito IBM
* Update Comparison - il sistema si occupa di confrontare gli aggiornamenti da eseguire e li seleziona
* Update Options - lasciare entrambe le opzioni in bianco e premere Next
* Update Execution - lo script si occupa di installare gli aggiornamenti
* Finish - premere finish e riavviare

 

La macchina si riavvierà più volte, lasciarla fare senza toccare niente, finché non ripartirà il sistema.

 

NOTE: Se non dovessero funzionare i pacchetti UXSP, è possibile ripetere la procedura specificando di scaricare i normali aggiornamenti al punto "Update Type".
