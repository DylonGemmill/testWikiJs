Esportare la VM in modalità OVF 1.0. Alcuni elementi di Virtualbox non sono però riconosciuti da VMWare, che in fase di importazione potrebbe restituire errori come:

Unsupported hardware family 'virtualbox-2.2'.

Il file OVA generato dall'esportazione è in realtà un file tar, che si può scompattare con

tar xvf nome_vm.ova -C OVF_TMP

all'interno della directory OVF_TMP ci sarà un file vmdk ed un ovf con la definizione della VM. 

Scaricare lo script Python allegato alla KB ed eseguirlo passando come parametro il file ovf. Sostituire il file vmx_ sul file ovf originale, ricompattare il file OVA ed importarlo in VMWare. 

Per ricompattare:

tar cvf ../nome_vm_v2.ova

Altre cose importanti:

  1. non devono esserci dispositivi IDE, solo SCSI
