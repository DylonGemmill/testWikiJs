Scaricare l'rpm di monitoraggio dal sito hp, installare alien ed eseguire:

apt-get -y install alien ia32-libs
alien --to-tgz hpacucli-8.50-6.0.noarch.rpm
mv opt/compaq /opt/
mv usr/sbin/* /usr/sbin/

 

Per entrare nella shell del programma di monitoraggio eseguire

hpacucli

 

I comandi principali per vedere lo stato del raid sono:

  • ctrl all show
    mostra l'elenco dei controller
  • ctrl all show detail
    mostra lo stato del raid. La riga che interessa principalmente è 
       Controller Status: OK
     
E' possibile controllare lo stato del raid con il plugin nagios allegato.
