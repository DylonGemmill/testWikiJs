Il bonding di due interfacce di rete consente una sorta di alta disponibliità della connessione di rete, così che se un cavo di rete o una scheda fallisce il server mantiene la connessione.

Come prima cosa installare ifenslave:

apt-get install ifenslave-2.6

 

Quindi modificare /etc/network/interfaces come segue:

auto eth0
iface eth0 inet manual
  bond-master bond0

auto eth1
iface eth1 inet manual
  bond-master bond0

auto bond0
iface bond0 inet static
  address 192.168.1.10
  gateway 192.168.1.1
  netmask 255.255.255.0
  bond-mode active-backup
  bond-miimon 100
  bond-lacp-rate 1
  bond-slaves none

associare le singole schede al bond invece che configurarle nel bond stesso (bond-slaves) velocizza l'attivazione dell'interfaccia.
