Tra i vari protocolli, l'ix2 supporta anche rsync per effettuare i backup. Andare in Rete > Protocolli ed assicurarsi che rsync sia abilitato. Se si eseguiranno backup su rete non protetta, è possibile limitare l'accesso alle cartelle di rsync premendo sul pulsante con l'ingranaggio per impostare una password di sincronizzazione.

 

Vengono messe a disposizione su rsync tutte le condivisioni create tramite Storage > Shares.

 

Per sincronizzare una directory da Linux alla condivisione Backups predefinita, usare il comando:

rsync -r /dati IP_IOMEGA::Backups

 

se si è abilitata l'autenticazione rsync:

rsync -r /dati rsync@IP_IOMEGA::Backups

e verrà chiesta la password configurata. Se si vuole evitare il prompt è possibile scriverla come prima riga di un file di testo ed usare il parametro --password-file=NOME_FILE_CON_PW.
