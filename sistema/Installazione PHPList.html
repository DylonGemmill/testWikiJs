Installazione

Scompattare phplist in una directory di lavoro, entrarci ed eseguire
mv public_html/lists /var/www/mailing
cd /var/www/mailing
chown www-data:www-data .
mkdir uploadimages
chmod 755 uploadimages
Quindi occorre creare il database per PHPList:
mysql -u root -p
mysql> create database phplist;
mysql> GRANT SELECT, INSERT, UPDATE, DELETE, CREATE ON phplist.* TO 'listuser'@'localhost' IDENTIFIED BY 'miapassword';
modificare il file config/config.php in particolare:
$database_host = "localhost";
$database_name = "phplist";
$database_user = "listuser";
$database_password = 'l10fFD25';
$installation_name = 'Mailing CIA Automazione';
$pageroot = '/mailing';
$adminpages = '/mailing/admin';
define ("REGISTER",0);
define("UNSUBSCRIBE_JUMPOFF",1); // se a 1 non richiede una ragione per l'unsubscribe
define("FCKIMAGES_DIR","uploadimages");
define ("TEST",0);
Collegarsi al server web dove abbiamo installato PHPList, sottodirectory admin/ per iniziare la configurazione:
http://webserver/mailing/admin/
Come prima operazione modificare la password di admin.
Andare in Configura  ed impostare:
  • indirizzo internet: url (senza http://) dell'installazione di phplist, per comporre gli indirizzi di unsubscribe ed altro
  • nome del dominio: dominio dell'indirizzo email
  • amministratore del server: indirizzo email (va bene lasciare nome@[DOMAIN])
  • chi riceve i report: email destinatario che riceverà i rapporti di invio
  • nome mittente delle email di sistema: impostare a piacimento
  • footer predefinito per i messaggi email: impostare il piè di pagina predefinito
Andare quindi nella pagina liste e creare una nuova lista.
