Installazione di nullmailer, per la consegna della posta 

 

sSMTP è un semplice pacchetto che simula la presenza di un MTA sulla macchina e consegna la posta ad un server SMTP.

Per installarlo eseguire il comando 

 

aptitude install ssmtp

 

Quindi aprire il file di configurazione /etc/ssmtp/ssmtp.conf e modificare i seguenti parametri:

  • root=postmaster (da COMMENTARE)
  • mailhub=nome del server SMTP (dipende dalla connessione)
  • hostname=mydomain.it (dominio da cui si vuol far provenire la mail, per evitare che venga rimbalzata dai mailserver)

Siccome l'SMTP è dipendente dalla connessione si può usare il mailserver di YetOpen che consente il relay tramite username e password. Per fare questo, modificare sempre ssmtp.conf con i seguenti parametri:

  • mailhub=uffizio.dyndns.org
  • AuthUser=username zimbra
  • AuthPass=password zimbra
