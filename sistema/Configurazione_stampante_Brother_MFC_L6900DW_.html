I driver della stampante sono scaricabili dal sito Brother. Includono però degli eseguibili a 32 bit, quindi servono le librerie di compatibilità sui sistemi a 64bit!

Prima di installare i due DEB scaricati da Brother eseguire

apt-get install gcc-multilib

Anche se file riporta i dati sul binario ldd non riesce a fare la verifica delle librerie collegate e lanciando il file il sistema dice che non esiste.

# file /opt/brother/Printers/MFCL6900DW/lpd/rawtobr3
/opt/brother/Printers/MFCL6900DW/lpd/rawtobr3: ELF 32-bit LSB executable, Intel 80386, version 1 (SYSV), dynamically linked (uses shared libs), for GNU/Linux 2.2.0, not stripped
# ldd /opt/brother/Printers/MFCL6900DW/lpd/rawtobr3
        non è un eseguibile dinamico
# /opt/brother/Printers/MFCL6900DW/lpd/rawtobr3
-bash: /opt/brother/Printers/MFCL6900DW/lpd/rawtobr3: File o directory non esistente

Attivando il debug di CUPS esce:

D [09/Dec/2019:09:32:51 +0100] [Job 392169] sh: 1: /opt/brother/Printers/MFCL6900DW/lpd/rawtobr3: not found
