csync è un tool per tenere in sincronia due directory su due sistemi remoti. E' in grado di gestire i conflitti e di mantenere l'ultimo file modificato.

E' disponibile un PPA per Debian e Ubuntu, che contiene il programma e le sue dipendenze. Eseguire:

apt-add-repository ppa:markhannon/ppa
apt-get update
apt-get install csync
