Icinga è un software di monitoraggio dei server basato su Nagios. E' web-based, quindi richiede la presenza di un server web per la gestione.

Ci sarà un server centrale di raccolta dati, e diversi server da monitorare (che chiameremo impropriamente client).

 

Sul server:

apt-get install icinga icinga-doc icinga-phpapi
  • Quando viene richiesta l'autoconfigurazione del browser sleezionare Apache2, così che venga creato il link in /etc/apache2/conf.d;
  • Password: selezionare una password per l'utente icingaadmin via web;
  • Gestione del database di icinga con dbconfig: No.
Al termine del wizard ci si può collegare a
http://nomeserver/icinga
verranno chiesti user e pass: l'utente è icingaadmin, la password quella fornita durante la configuazione del pacchetto.
Sarà già presente un host localhost con alcuni servizi predefiniti, come il carico di sistema e il controllo sull'occupazione del disco.
Come prima cosa andiamo a personalizzare la configurazione. 
Modificare il file /etc/icinga/objects/contacts_icinga.cfg 
rif: http://www.howtoforge.com/server-monitoring-with-icinga-on-ubuntu-11.10
