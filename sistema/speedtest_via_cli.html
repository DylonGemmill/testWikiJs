Con wget/cURL

Un modo più rozzo è quello di scaricare un file di grosse dimensioni da un sito che si assume abbastanza veloce usando cURL o wget:

wget http://ubuntu.mirror.garr.it/ubuntu-releases/18.04.3/ubuntu-18.04.3-desktop-amd64.iso -O /dev/null
curl -o /dev/null http://ubuntu.mirror.garr.it/ubuntu-releases/18.04.3/ubuntu-18.04.3-desktop-amd64.iso
