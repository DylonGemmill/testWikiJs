IPMI è l'interfaccia per la gestione ed il controllo delle funzionalità di un server.

 

apt-get install ipmitool
echo -e "ipmi_msghandler\nipmi_devintf\nipmi_si" >> /etc/modules

Per verificare il funzionamento:

modprobe ipmi_msghandler
modprobe ipmi_devintf
modprobe ipmi_si
ipmitool sel

 

Interrogazione dei sensori

Per interrogare i sensori di temperatura, voltaggi e ventole:

ipmitool sdr elist full

 

Per vedere le temperature:

ipmitool sdr type temperature

 

Per l'elenco degli eventi:

ipmitool sel list

 

Un riepilogo dello stato hardware si può avere con

ipmitool chassis status

Configurazione accesso remoto di IMPI

Di default c'è un utente amministrativo. Per vederlo digitare il comando

ipmitool user list 1

per cambiare nome utente e password:

ipmitool user set name 2 ipmiuser
ipmitool user set password 2 ipmipass

per un bug in impitool l'utente deve avere i privilegi di amministratore, quindi (1 è il canale e dovrebbe essere sempre 1, 2 è l'id dell'utente sopra)

ipmitool channel setaccess 1 2 privilege=4

rifacendo il comando sopra il nome all'id 2 sarà aggiornato. Configurare quindi l'interfaccia di rete di gestione:

ipmitool lan set 1 ipsrc static
ipmitool lan set 1 ipaddr 10.22.22.135
ipmitool lan set 1 netmask 255.255.255.0

Per verificare lo stato della rete;

ipmitool lan print
