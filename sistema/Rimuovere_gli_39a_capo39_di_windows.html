Per rimuovere i fastidiosi a capo di windows (visualizzati come ^M) torna utile il programma dos2unix che 'depura' i file da questi fastidiosi caratteri.

 

dos2unix nomefile.txt

 

Il comando modifica il file cancellando i caratteri indesiderati.

Per depurare un'intera cartella di sorgenti:

 

find mysrc/ -iname "*.php" -exec dos2unix {} \;
