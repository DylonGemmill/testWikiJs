In questo esempio configuriamo 4 wlan sulla scheda di rete eth1.

 

Eseguire questi comandi:

apt-get install -y vlan
modprobe 8021q
echo "8021q" >> /etc/modules
vconfig add eth1 10
vconfig add eth1 20
vconfig add eth1 30
vconfig add eth1 40

Quindi modificare /etc/network/interfaces in questo modo, per la parte relativa ad eth1:

auto eth1

auto vlan40
iface vlan40 inet manual
vlan-raw-device eth1

auto vlan20
iface vlan20 inet manual
vlan-raw-device eth1

auto vlan30
iface vlan30 inet manual
vlan-raw-device eth1

auto vlan10
iface vlan10 inet manual
vlan-raw-device eth1

A questo punto possiamo eseguire

/etc/init.d/networking restart

per rendere effettive le modifiche ed ottenere le interfacce in Linux.
