Scaricare l'ultimo pacchetto da nagiosql.org e scompattarlo nella directory del web server.

Opzionalmente dipende da php+ssh, se lo vogliamo:

apt-get install libssh2-php

Poi occorre disabilitare il flag suhosin.session.encrypt di php. Si può modificare nei file .ini di php.

Quindi creare la directory per ospitare i file di config di nagiosql:

mkdir /etc/nagiosql
chown www-data:www-data /etc/nagiosql

Riavviare apache, quindi aprire il browser alla posizione appena scompattata, che sarà http://SERVER/nagiosql32. Partirà l'installer, che crea già da solo il database. Se non compare il tipo di database (primo campo, inibito) ripartire con l'installer da index.php.

Proseguire con tutti i passaggi, e quando viene chiesto rimuovere la sottocartella install/
