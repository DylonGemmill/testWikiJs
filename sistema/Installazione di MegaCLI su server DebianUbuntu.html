Esistono dei repo che contengono il comando di amministrazione del raid, e degli script di monitoraggio dello stato del sistema.

Creare un file /etc/apt/sources.list.d/megaraid.list contenente

deb http://hwraid.le-vert.net/distrib branch main

ATTENZIONE, distrib deve essere sostituito con debian o ubuntu, mentre branch con il nome della release.

Quindi eseguire

apt-get update && apt-get install megacli

 

Per installare la parte di monitor

apt-get install megaclisas-status

quindi creare il file /etc/default/megaclisas-statusd con all'interno

MAILTO=support@yetopen.it   # Email che riceverà le notifiche
PERIOD=600    # Secondi tra ogni controllo
REMIND=7200   # Secondi tra ogni notifica (default 2h)

 

Rif: http://hwraid.le-vert.net/wiki/DebianPackages
