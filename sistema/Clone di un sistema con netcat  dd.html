Sul server di destinazione eseguire:

 

nc -l -p 9000 | dd of=/dev/sda bs=16065b

 

-p è la porta dove sarà in ascolto netcat, usare una porta libera/dev/sda è il device di destinazione!

Sul server sorgente:

 

dd if=/dev/sda bs=16065b | nc 192.168.0.254 9000


/dev/sda è il device sorgente!192.168.0.254 è l'ip del server di destinazione, con la porta specificata sopra con il parametro -p.

 

In ogni momento è possibile sapere lo stato di avanzamento della copia. Identificare il PID di dd e quindi lanciare il comando 

kill -USR1 pid_processo

e dd stamperà le statistiche di trasferimento fino a quel momento senza interrompersi.
