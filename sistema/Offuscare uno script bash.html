Usare lo script da questa pagina:

http://www.datsi.fi.upm.es/~frosal/

 

Va estratto e compilato con un semplice make. La compilazione però fallisce perché in Ubuntu c'è un controllo sul ptrace (?). Per farlo andare:

echo 0 > /proc/sys/kernel/yama/ptrace_scope

Per eseguire l'obfuscation:

shc -f nomescriptbash

verrà creato un file nomescriptbash.x ELF-binary con la stessa funzione.

 

Per compilare un eseguibile a 32bit su un sistema a 64 lanciare in questo modo:

CFLAGS="-m32" shc -f nomefile
