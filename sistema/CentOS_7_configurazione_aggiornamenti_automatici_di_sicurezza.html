In breve:

yum -y install yum-cron
sed -i 's/^\(update_cmd\s*=\s*\).*$/\1security/' /etc/yum/yum-cron.conf
sed -i 's/^\(download_updates\s*=\s*\).*$/\1yes/' /etc/yum/yum-cron.conf
sed -i 's/^\(apply_updates\s*=\s*\).*$/\1yes/' /etc/yum/yum-cron.conf
systemctl start yum-cron
systemctl enable yum-cron

Per configurare le notifiche via email degli aggiornamenti modificare questi parametri:

emit_via = email
email_from = root@hakase-labs
email_to = myemail@gmail.com
email_host = hakase-labs
