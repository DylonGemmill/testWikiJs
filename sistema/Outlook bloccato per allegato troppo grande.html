Situazione:

inviando una mail con Outlook con allegato troppo grande, il sistema windows si blocca.

 

Soluzione:

1. Fermare il sistema di posta. Solitamente è installato postifix, quindi fermare con il comando

/etc/init.d/postfix stop

2. Riavviare il pc windows

3. Riaccendere Outlook

4. Eliminare la mail che blocca il sistema

5. Riavviare il sistema di posta con il comando

/etc/init.d/postfix start
