Il comando mail, del pacchetto bsd-mailx, è molto comodo ma non permette di per sé l'invio di allegati. Per poter inviare file è necessario appoggiarsi al comando uuencode, del pacchetto sharutils.

Supponendo di dover inviare il file /root/backup.bz2, aggiungendo come testo alla mail "Ecco il file di log", procediamo come segue:

echo "Ecco il file di log" > /tmp/mailtmp
uuencode /root/backup.bz2 /root/backup.bz2 >> /tmp/mailtmp
cat /tmp/mailtmp | mail -s "Oggetto della mail" destinatario@yetopen.it
