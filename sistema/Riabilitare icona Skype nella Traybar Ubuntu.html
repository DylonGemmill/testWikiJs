Prima della 13.10 era possibile agire nel registro di gnome con il comando

gsettings set com.canonical.Unity.Panel systray-whitelist "['JavaEmbeddedFrame', 'Wine', 'Update-notifier', 'Skype']"
oppure
gsettings set com.canonical.Unity.Panel systray-whitelist "['all']"

 

Dalla 13.10 in poi, invece, la voce nel registro di gnome non è più presente e con la versione di skype scaricata dal sito si continua ad avere il problema. Per risolvere occorre abilitare il repo dei partner e scaricare skype da lì

sudo add-apt-repository "deb http://archive.canonical.com/ $(lsb_release -sc) partner"
sudo apt-get update
sudo apt-get install skype && sudo apt-get -f install
