Scaricare da rdiffbackupweb.sourceforge.net il tgz e scompattarlo in una directory accessibile via web.

Sempre sul server:

apt-get install python-mysqldb

ciao

collegarsi alla shell di mysql ed eseguire:

create database rdiffdb;
grant all on rdiffdb.* to rdiffuser@'%' identified by 'password';
flush privileges;

quindi dalla directory scompattata

cat rdiff.sql  | mysql -u rdiffuser -p rdiffdb
cp process_backups.py /etc/cron.daily/

aprire il file config.php, modificare i parametri di connessione mysql in fondo, ed i percorsi delle directory (BACKUP_STORE_DIR e vicine).

Ora è possibile andare tramite web al sito dove si è installato il software ed effettuare le prime configurazioni. Entrare con admin/admin.
