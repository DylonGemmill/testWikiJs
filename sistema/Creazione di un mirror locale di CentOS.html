Creare una directory su uno storage con capacità sufficiente per ospitare il mirror.

Come prima cosa, se disponibile preleviamo i pacchetti disponibili nel DVD di installazione.

mkdir centos-mi.mirror.garr.it/x86_64
mount -o loop CentOS-6.2-x86_64-bin-DVD1.iso /mnt/tmp
rsync -avHPS /mnt/tmp/ centos-mi.mirror.garr.it/x86_64/

 

quindi lanciare un comando come il seguente:

rsync -avrt rsync://mi.mirror.garr.it/CentOS/6.2/os/x86_64 destinazione/

 

in questo caso vogliamo creare il mirror per la versione 6.2 amd64. Una volta effettuato il primo clone provvediamo a schedulare con cron lo stesso comando giornalmente o settimanalmente.
