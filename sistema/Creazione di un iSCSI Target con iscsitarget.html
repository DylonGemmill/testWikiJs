Come prima cosa installare il pacchetto principale

apt-get install iscsitarget
sed -i 's/ISCSITARGET_ENABLE=false/ISCSITARGET_ENABLE=true/' /etc/default/iscsitarget

se compare un errore che manca il modulo iscsi_trgt installare anche il pacchetto relativo

apt-get install iscsitarget-dkms

 

Modificare il file /etc/iet/ietd.conf aggiungendo in fondo una configurazione come questa:

Target iqn.2012-11.it.yetopenlab:storage.lvmiscsi
        IncomingUser utente password
        OutgoingUser
        Lun 0 Path=/dev/VolGroup/LVName,Type=fileio,ScsiId=23456765432345

in questo caso stiamo condividendo il volume LVName nel gruppo VolGroup.

Riavviare il servizio iscsitarget ed avremo una condivisione iscsi disponibile sulla rete.
