Mi è capitato sull'HP Proliant ML350 di Eco86 che l'installazione via USB si bloccasse durante l'installazione base del sistema.

Per risolvere ho trovato questo workaround: appena iniziata l'installazione aprire una shell sul secondo terminale ed eseguire questo comando:

while true; do rm /var/run/chroot-setup.lock; sleep 1; done

 

In questo modo funziona.
