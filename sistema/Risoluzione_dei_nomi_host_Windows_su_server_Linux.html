In caso di server Samba, smbclient è in grado di risolvere i nomi degli host Windows, ma Linux no. Ovvero, facendo

smbclient -L //hostwindows

funziona, ma 

ping hostwindows

no. Per fare in modo che funzioni:

aptitude install winbind

modificare quindi il file /etc/nsswitch.conf, ed alla riga hosts aggiungere wins prima di dns, come nell'esempio:

hosts:          files wins dns
