Con oC è possibile condividere una directory del filesystem, con alcuni accorgimenti.

  1. Aggiungere l'utente apache (o www-data in Ubuntu) al gruppo delle condivisioni, ad esempio:
    useradd -G ufficio apache
  2. verificare che sulla directory ci sia attivo lo sticky bit sul gruppo
  3. siccome la umask di apache è diversa i file vengono creati come apache:ufficio ma con permessi rwxr-----. Per forzare i permessi corretti usiamo le ACL:
    setfacl -R -d --set u::rwx,g::rwx,o::- /dati/percorso_da_forzare
