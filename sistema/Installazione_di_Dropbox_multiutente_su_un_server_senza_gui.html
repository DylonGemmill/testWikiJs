Dropbox ha una parte demone ed una parte GUI. Sui server è possibile installare solo la prima parte.

 

Collegarsi con l'utente per cui si vuole registrare l'account quindi eseguire:

cd ~ && wget -O - "http://www.dropbox.com/download?plat=lnx.x86_64" | tar xzf -
cd ~ && wget -O - "http://www.dropbox.com/download?plat=lnx.x86" | tar xzf -
~/.dropbox-dist/dropboxd 

 

A questo punto verrà fornito un url per la verifica, copiarlo ed incollarlo nel browser per eseguire login. Effettuata questa operazione l'account sarà collegato, e si potrà premere CTRL-C per terminare l'esecuzione manuale del demone.

 

Per rendere l'avvio automatico, copiare lo script presente in yo_utils.git/serverutils/ MODIFICANDO la riga DROPBOX_USERS con l'elenco degli utenti per cui dovrà girare il demone, separati da spazio.

Quindi per rendere eseguibile lo script ed avviato in automatico:

chmod +x /etc/init.d/dropbox
update-rc.d dropbox defaults
