Con find è possibile trovare un file che è stato modificato entro -mtime n*24 ore precedenti. Non è però possibile specificare un orario più preciso. Per fare questo, si usa una escamotage: find può usare un file come confronto, quindi

touch file_prova -t AAAAMMGGHHMM

find . -newer file_prova

In questo modo find troverà tutti i file più recenti di file_prova, e quindi della data assegnatagli.

Per trovare i file entro un range, si crea un secondo file e il comando sarà il seguente:

find . -newer file_da -and ! -newer file_a

ovviamente newer si possono usare con i modificatori cnewer (data di creazione) e anewer (data di ultimo accesso).
