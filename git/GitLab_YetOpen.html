Gestione runners

Servono per i test automatici di CI/CD.

Sono installati tramite Docker sulla VM code.ufficyo.com.

L'installazione è stata fatta con questa guida. La registrazione è stata fatta con questo comando:

docker exec gitlab-runner gitlab-runner register --url https://code.ufficyo.com --registration-token TOKEN --non-interactive --name code-docker-01 --executor shell

per creare il runner code-docker-01.
