Capita a volte di fare le modifiche dell'ultimo minuto senza controllarle. 

Git ha gli hooks, dei comandi che possono essere richiamati automaticamente prima o dopo un evento. Ad esempio si può eseguire uno script prima di un commit.

Per effettuare un controllo sintattico sui file php modificati del progetto si può usare questo pre-commit hook script:

https://github.com/ReekenX/git-php-syntax-checker

Per installarlo basta semplicemente copiarlo come pre-commit nella directory ROOT_PROGETTO/.git/hooks/ .

 

Ad esempio posizionandosi nella root del progetto eseguire:

wget https://raw.github.com/ReekenX/git-php-syntax-checker/master/pre-commit -P .git/hooks/ ; chmod +x .git/hooks/pre-commit
