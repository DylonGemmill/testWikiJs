Per annullare un commit sbagliato mantenendo le modifiche ai file:

$ git reset --mixed HEAD~1

Per annullare invece il commit scordando le modifiche:

$ git reset --hard HEAD~1

Modificando il numero finale, per esempio HEAD~2, HEAD~3, ecc..., si possono annullare più commit in una sola volta.

Per portare il cambiamento al branch remoto bisogna forzare il push (git push -f) altrimenti dà errore e non funziona. In questo caso, assicurarsi assolutamente che nessuno abbia pullato i commit che si stanno rimuovendo.
