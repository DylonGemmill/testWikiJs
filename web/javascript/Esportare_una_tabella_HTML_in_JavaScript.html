Un plugin di jQuery consente di esportare la tabella di una pagina HTML in un file CSV, e di far partire automaticamente il download.
Ho modificato il plugin standard per far partire il download in automatico (in allegato).

Per usufruire della funzione, includere jQuery ed il plugin nella sezione <HEAD> dell'html:

 

<script src="jquery.js" type="text/javascript"></script>
<script src="jquery.table2csv.js" type="text/javascript"></script>

 

Assicurarsi che la tabella abbia un identificativo univoco per la ricerca tramite jQuery, e che il corpo della tabella sia incluso nel tag <TBODY>.
Indispensabile per il download automatico la predisposizione di un div di nome main nella pagina. Il nome del div è hardcodato nello script table2csv.js.
Ad esempio:

 

<table id="miatabella">
 <thead>
  <tr>
   <th>Intestazione 1</th>
   <th>Intestazione 2</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <th>Dato 1</th>
   <th>Dato 2</th>
  </tr>
 </tbody>
</table>

<!-- questo div è indispensabile per il download in automatico -->
<div id="main"></div>

 

E' possibile offire il download della tabella con un pulsante:

 

<input value="Download Excel" type="button"   onclick="$('#miatabella').table2CSV({separator:';'});">
