Lavorando con id contenti caratteri particolari, ho scoperto che:

$("li#ZZBLINDO10/AGR")

non funziona. Rimuovendo il carattere '/', invece, funziona. Probabilmente il selettore # di JQuery non riesce a gestire un set particolare di caratteri, tra cui '/'. Per ovviare al problema occorre trattare l'id come semplice attributo, utilizzando quindi il seguente selettore:

$("li[id=ZZBLINDO10/AGR]")
